Directions

In this assignment you are going to complete the implementation of a collection class named LinkedQueue that implements a Queue ADT. The Queue ADT requirements are outlined in the following Java interface.

public interface Queue<E>
{
    // adds an item to the back of the queue
    public void enqueue(E item);

    // removes an item from the front of the queue
    public E dequeue();

    // returns the front item in the queue without removing it
    public E front();

    // returns true if the queue is empty; otherwise returns false
    public boolean isEmpty();

    // returns a String representation of the queue
    public String toString();

}

The LinkedQueue class will be implemented using a static data structure. The class will also incorporate Java generics.
