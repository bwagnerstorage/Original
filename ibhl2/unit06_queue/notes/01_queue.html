Queue

A queue is a FIFO (First-In-First-Out) data structure. Data enters the back of the queue and exits the front.

The illustration below demonstrates the functionality of a queue.

Queue Implementation

A queue can be implemented as either a static data structure(array) or a dynamic data structure(object reference). Although a queue can be implemented with an array the dynamic approach is a better choice because of the fact that data enters the data structure from the back and exits from the front. With the array implementation a removal from the front of the array would require data movement to fill the vacant position in the array. The array approach also has the issue of a fixed size.

As with a stack, the dynamic data structure approach requires the use of another class to create the nodes.

public class ListNode
{
   // Instance Variables
   private Object value;
   private ListNode next;

   // Constructor
   public ListNode(Object initValue, ListNode initNext)
   {
      value = initValue;
      next = initNext;
   }

   // Accessor Methods
   public Object getValue()
   {
      return value;
   }

   public ListNode getNext()
   {
      return next;
   }

   // Mutator Methods
   public void setValue(Object theNewValue)
   {
      value = theNewValue;
   }

   public void setNext(ListNode theNewNext)
   {
      next = theNewNext;
   }
}
