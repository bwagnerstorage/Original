Write a Java program that uses a hash table to implement a spelling checker. The program starts by reading a list of words from a data file and inserts them into a hash table using the hash function shown below. Chaining should be used to resolve collisions. Once the hash table has been populated with words allow the user to enter words and have the program search the table for the input words. If a word is found in the table notify the user that the word was spelled correctly. If a word is not found in the table notify the user that the word was spelled incorrectly.

// The hash function
// Returns a valid index within the range of arrays indices
public int hashCode(String key)
{
    int sum = 0;
    for(int i=0; i < key.length(); i++) 
    {
       sum += (int)key.charAt(i);   
    }
    return sum % TABLE_SIZE;
}

Directions

  1. Implement the method add of the SpellChecker class. This method should insert the given word into the hash table using chaining to resolve any collisions that might occur. The method add should use the method hashCode to retrieve the initial insertion point for the given word. If the array location given by the call to hashCode is already occupied the method should traverse the linked list at that location and append the word to the end of the list (chaining).

  2. Implement the method search of the SpellChecker class. This method should search the hash table for the given word, if the word is found in the table the method should return true, otherwise it should return false.
