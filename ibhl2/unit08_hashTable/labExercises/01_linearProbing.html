Linear Probing

Directions

Write a Java program that demonstrates the Linear Probing collision resolution strategy.

Your program needs to do the following:

  1. Create an integer array (hash table) size 12.

  2. Insert random numbers in the range of 10 to 99 into the array using the following hash function:

    public int hash(int data)
{
   return data % table.length;    // table is the name of the array
}

  3. When a collision occurs use linear probing to resolve the collision. Include the following code in your implementation:

    while(table[index] > 0)  // while not an empty cell
{
   index = (index + 1) % table.length;  // advance to next cell, 
                                        // includes wrap around
}
table[index] = num;  // new insertion point found, insert data

  4. Display the entire table every time a number is added to the table. Pause the program between insertions.

    keyboard.nextLine();  // pause

  5. Include a message that shows which index each number is inserted in. If a collision occurs display a message that informs the user that linear probing was activated to resolve the collision. (Refer to the sample output below)

  6. Add numbers to the table until the table is full.

  7. Once the table is full, perform searches looking for certain numbers within the table. When a number is found display the array index where the number was found. If the number was not found display a message indicating that the number is not in the list.
