LinkedList3

Directions

Write a program that inserts 25 random integers into a linked list. The numbers should be added to the list so that it maintains a sorted order. The random numbers should be in the range of 0 to 100.

Below is a complete list of all the methods in the IB Java Examination Tool Subset (Jets).
