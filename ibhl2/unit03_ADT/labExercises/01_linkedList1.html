Helpful Information

In section 2 of the notes on lists you learned that both the ArrayList class and the LinkedList class implement the List Interface and therefore have the following methods in common:

  • size()
  • boolean add(E e) // append to end of list
  • add(int index, E e) // insert at position index within list
  • E get()
  • E set(int index, E e)
  • E remove(int index)
  • Iterator <E> interator()
  • ListIterator <E> listIterator()

If you look at the method signature for Java's LinkedList class you will notice something interesting. Java classes can implement more than one interface.

public class LinkedList<E> implements List<E>, Deque<E>  
// other interfaces not shown

These interfaces require the LinkedList class to provide more functionality. The Deque<E> interface provides additional functionality for the LinkedList class that the IB curriculum expects IB students to be able to use in solving programming problems involving the LinkedList class. These methods include the following:

  • addFirst(E e)
  • addLast(E e)
  • clear()
  • element()
  • E getFirst()
  • E getLast()
  • E remove()
  • E removeFirst()
  • E removeLast()
  • boolean isEmpty()

Below is a complete list of all the methods in the IB Java Examination Tool Subset (Jets).

Method Description
add(E e) Appends the specified element to the end of this list.
add(int index, E element) Appends the specified element to the end of this list.
addFirst(E element) Inserts the specified element at the front of this list.
addLast(E element) Inserts the specified element at the end of this list.
clear() Removes all of the elements from this list.
E element() Retrieves, but does not remove, the first element of this list.
E get(int index) Returns the element at the specified position in this list.
E getFirst() Retrieves, but does not remove, the first element of this list.
E getLast() Retrieves, but does not remove, the last element of this list.
boolean isEmpty Returns true if this list contains no elements.
Iterator iterator() Returns an iterator over the elements in this list in proper sequence.
ListIterator listIterator() Returns a list iterator over the elements in this list in proper sequence.
E remove() Retrieves and removes the first element of this list.
E remove(int index) Removes the element at the specified position in this list.
E removeFirst() Retrieves and removes the first element of this list.
E removeLast() Retrieves and removes the last element of this list.
int size() Returns the number of elements in this list.

Copy the following code to a java file named Location.

public class Location
{
   // instance variables
   private int x;
   private int y;
            
   // constructor
   public Location(int x, int y)
   {
      this.x = x;
      this.y = y;
   }
            
   // accessor methods (getters)
   public int getX()
   {
      return x;
   }
            
   public int getY()
   {
      return y;
   }
            
   // mutator methods (setters)
   public void setX(int x)
   {
      this.x = x;
   }
            
   public void setY(int y)
   {
      this.y = y;
   }
            
   // toString method
   public String toString()
   {
      return "(" + getX() + "," + getY() + ")";
   }
}

Method linkedList1

Complete method linkedList1 using the following instructions.

  1. Create a LinkedList object using the following code:
    LinkedList<Integer>list = new LinkedList<>();
  2. Use the add method to append the number 10 and the number 20 to the end of list.
  3. Use the add method to insert the number 5 at index 0 of list.
  4. Use the add method to insert the number 15 at index 2 of list.
  5. Use the get method to retrieve the first element in list and display it using the following format: 
    first = <number>
  6. Use the get method and the size method to retrieve the last element in list and display it using the following format: 
    last  = <number>
  7. Use an enhanced for loop to display the elements in list horizontally with a space between each one.
  8. Use the remove method to remove the second element in list.
  9. Use another enhanced for loop to display the remaining elements in list horizontally with a space between each one.

Method linkedList2

Complete method linkedList2 using the following instructions.

  1. Create a LinkedList object using the following code:
    LinkedList<String>list = new LinkedList<>();
  2. Use the addFirst method to add the string "elephant" to the beginning of list.
  3. Use the addLast method to add the string "lion" to the end of list.
  4. Use the addFirst method to add the string "cheetah" to the beginning of list.
  5. Use the addLast method to add the string "chimpanzee" to the end of list.
  6. Use the addFirst method to add the string "giraffe" to the beginning of list.
  7. Use the getFirst method to retrieve the first element in list and display it using the following format: 
    first = <string>
  8. Use the getLast method to retrieve the last element in list and display it using the following format: 
    last  = <string>
  9. Use the following code to access the Iterator object in list: 
    Iterator iter = list.iterator();
  10. Use the Iterator methods hasNext and next along with a while loop to display all of the elements in list horizontally with a space between each one.
  11. Use the method removeFirst to remove the first element in list.
  12. Use the method removeLast to remove the last element in list.
  13. Use the Iterator methods hasNext and next along with a while loop to display the remaining elements in list horizontally with a space between each one.
  14. Use the element method to retrieve the first element in list and display it using the following format: 
    first = <string>
  15. Use the method clear to remove all of the elements in list.
  16. Use the method isEmpty to display the string "List is empty" if list contains no elements; otherwise display the string "List is not empty".

Method linkedList3

Complete method linkedList3 using the following instructions.

  1. Create a LinkedList of Location objects using the following code:
    LinkedList<Location>list = new LinkedList<>();
  2. Use the addLast method to add the location (1,1) to the end of list.
  3. Use the addLast method to add the location (2,2) to the end of list.
  4. Use the addLast method to add the location (3,3) to the end of list.
  5. Use the addFirst method to add the location (0,0) to the beginnning of list.
  6. Use the getFirst method to retrieve a Location object from the beginning of list and store its reference in a Location variable name loc.
  7. Use the Location class methods setX and setY to modify the Location object reference by loc so that its x and y values are set to 100.
  8. Use the Location class toString method, the Deque interface methods isEmpty and removeFirst, and a while loop to remove and display all of the elements in list.
