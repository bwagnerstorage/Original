There are three notations used to represent arithmetic expressions:

  • Infix Notation
  • Prefix Notation
  • Postfix Notation

Infix Notation

This is the notation you use in math class. In this notation an operator is positioned between two operands (Ex. 7 + 8)

Prefix Notation

In this notation an operator is positioned in front of two operands (Ex. + 7 8)

Postfix Notation

In this notation an operator is positioned after two operands (Ex. 7 8 +)

Both prefix and postfix notation are used in computers to evaluate arithmetic expressions. They use these notations instead of infix because parentheses are not needed to express precedence rules(PEMDAS).

The table below shows some arithmetic expressions in both infix and postfix notations.

Evaluating a Postfix Expression

Start at the left of the expression. Each time an operator is
encountered, replace the operator and its preceding two
arguments with the value calculated by applying the operator
to its preceding two operands (values). For example,

3 4 * 2 +    // replace 3 4 * with 12
   12 2 +    // replace 12 2 + with 14
       14    // the value of the expression

Another example:

3 4 2 * +    // replace 4 2 * with 8
    3 8 +    // replace 3 8 + with 11
       11    // the value of the expression

split Method

The String class has a useful method named split that you can use to simplify your implementation. Look at the following example: 

String exp = "12 8 +";
String[] tokens = exp.split(" ");

The split method breaks a string apart into tokens and stores them in an array of Strings. The space in the split's parameter list determines how the string is going to be split apart. In this case, a new token is created each time a space is encounter. After execution, the array tokens contains the following:

0 1 2
"12" "8" "+"
