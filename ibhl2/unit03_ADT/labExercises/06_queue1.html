Queue1

Directions

Write a program that allows you to enter and delete integers from a list. All additions to the list must occur at the end of the list and all deletions must occur at the beginning of the list (FIFO). Use the LinkedList class which implements the Deque interface. The program needs to be menu driven like the sample output below.

Each time an operation is performed on the queue the queue needs to be printed. The following method can be used to print the queue without performing a dequeue:

public void printQueue(DeQue<Integer> queue)
{
   System.out.println("Current List");
   for(Integer n : queue)
   {
   		System.out.print(n + " ");
   }
   System.out.println();
   System.out.println();
}
