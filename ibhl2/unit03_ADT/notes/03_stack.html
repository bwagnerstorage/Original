Stack ADT

A stack is a linear collection in which access is completely restricted to just one end, called the top. The classical example is the stack of food trays found in every cafeteria. Whenever a tray is needed, it is removed from the top of the stack, and whenever clean ones come back from being washed, they are again placed on the top. No one ever takes a tray from the middle of the stack, and it is even possible that trays near the bottom are never used. Stacks are said to adhere to a Last-In, First-Out protocol (LIFO). The last tray added to the stack is the first one taken.

Applications of Stacks

Applications of stacks in computer science are numerous. Following are just a few examples:

  • Parsing expressions in compilers
  • Translating infix expressions to postfix form and evaluating postfix expressions
  • Backtracking algorithms
  • Managing computer memory in support of method calls
  • Supporting the "undo" feature in text editors, word processors, spreadsheet programs, etc.
  • Maintaining a history of the links visited by a Web browswer

Java Collection Framework(JCF)

The JCF includes a Stack class. The methods used to manipulate a stack are shown below. The operations for putting items on and removing items from a stack are called push and pop, respectively.

class java.util.Stack<E> (partial)

E push(E item) pushes item onto the top of the stack; returns item
E pop() removes and returns the element at the top of the stack
E peek() returns the element at the top of the stack without removing it
boolean isEmpty() returns true if stack is empty; false otherwise
* Note: The Stack class provided by the JCF is a subclass of another class called Vector (similar to an ArrayList). An unfortunate side effect of this implementation is that clients can treat stacks as vectors by inserting, replacing, or removing an element at any position, thus violating the spirit of a stack which only allows manipulation from the top.

Using a Stack

Creating a Stack Object

Stack<Integer> myStack = new Stack<>();

or

List<Integer> myStack = new Stack<>();

Adding an Object to a Stack

myStack.push(57);

Removing an Object from a Stack

int num = myStack.pop();

Viewing the Top of a Stack

int num = myStack.peek();  // does not remove the item

Traversing a Stack

while(!myStack.isEmpty())
{
   System.out.println(myStack.pop());
}
