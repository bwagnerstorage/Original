Heap

A heap is a complete binary tree that satisfies the heap property.

Complete binary tree

A binary tree is complete if each level except the last has a complete complement of nodes and if the nodes on the last level are filled in from the left. More simply a complete binary tree is a tree where nodes are added from top to bottom, left to right, without leaving any spaces.

Heap Property

The heap property is the relationship that characterizes nodes and their children in a heap (e.g., each node is less than or equal to either of its children).

Minimum Heap Property (MinHeap)

- The value stored at each node is less than or equal to the values stored at its children.

Maximum Heap Property (MaxHeap)

- The value stored at each node is greater than or equal to the values stored at its children.
