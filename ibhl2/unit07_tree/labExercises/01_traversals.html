A common operation performed on a tree is a traversal. A tree traversal is the process of visiting (examining and/or updating) each node in a tree.

View the following illustration that demonstrates three traversals: preorder, inorder, and postorder.

Below is the pseudocode for a preorder traversal:

if NODE != null then
  process NODE
  visit NODE'S left subtree
  visit NODE'S right subtree
end if

The two methods below implement a preorder traversal recursively. The processing that occurs in the methods is to add each node's value to an ArrayList.

// preorder traversal : root-left-right

public List<Object> preOrderTraversal()
{
   List<Object> list = new ArrayList<>();
   preOrderTraversal(root, list);
   return list;
}
   
private void preOrderTraversal(TreeNode tree, List<Object> list)
{
   if( tree!= null)
   {
      list.add(tree.getValue());
      preOrderTraversal(tree.getLeft(), list);
      preOrderTraversal(tree.getRight(), list);
   }
}
