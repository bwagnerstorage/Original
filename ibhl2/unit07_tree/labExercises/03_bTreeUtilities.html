The method shown below returns true if all the values stored in a tree are less than the given value; otherwise it returns false.

// returns true if root is null or if all values stored in
// tree represented by root are less than key;
// otherwise returns false
public boolean valsLess(int key)
{
   return valsLess(root, key);
}

private boolean valsLess(TreeNode root, int key)
{
   if(root == null)
      return true;
   
   int value = (Integer)root.getValue();
   return value < key && valsLess(root.getLeft(), key) && valsLess(root.getRight(), key);
}

Directions

In this assignment you are going to write four binary tree methods: minValue, treeHeight, hasPathSum, and mirror.

  • minValue - returns the minimum data value found in the tree. This method should be implemented non-recursively.

  • treeHeight - returns the number of nodes along the longest path from the root node down to the farthest leaf node.

  • hasPathSum - returns true if there is a path from the root down to a leaf, such that adding up all the values along the path equals the given sum.

  • mirror - changes a tree so that the roles of the left and right pointers are swapped at every node.
