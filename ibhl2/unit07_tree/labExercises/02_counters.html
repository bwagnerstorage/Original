The method shown below counts the number of nodes in binary tree.

 
// counts and returns the number of nodes in tree

    public int numNodes() 
    { 
       return numNodes(root);
    }
    
    public int numNodes(TreeNode node) 
    { 
       if(node == null) 
       { 
          return 0; 
       } 
       else 
       { 
          return 1 + numNodes(node.getLeft()) + numNode(node.getRight()); 
       } 
    }

Directions

In this assignment you are going to write two methods: numLeaves and numParents. numLeaves should count the number of leaf nodes in a binary tree and numParents should count the number of parent nodes in a binary tree. Leaf nodes do not have any children and parent nodes have at least one child.
