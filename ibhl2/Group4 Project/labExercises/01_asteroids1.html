Asteroid Ship

The Asteroid ship only moves forward by applying thrust. If you want to go in a different direction the ship must be turned or rotated in the direction you want the ship to go.

Greenfoot Review

Greenfoot's Actor class provides two ways to move an Actor.

Option 1 (vector: magnitude and angle values)

Use the move and turn methods. By using these methods your are using a velocity vector where move represents the magnitude or length of the vector and turn represents the angle(in degrees)

Example

move(2);    // 2 pixels
turn(90);   // 90 degrees

Option 2 (vector: x and y values)

Use the getX, getY, and setLocation methods. By using these methods you are using a velocity vector as a (x,y) position on a coordinate plane. The method getX returns the Actor's current x location and getY returns the Actor's current y location. The setLocation method allows you to update an Actor's position by providing the destination location.

Example

setLocation(getX() + vx,  getY() + vy);

This example uses getX and getY to get the Actor's current location and then adds the horizontal velocity(vx) and the vertical velocity(vy) to move the Actor to its new position.

Conversion Formula's

Formula for converting the properties magnitude and direction into x and y properties.

ax = thrust * Math.cos(Math.toRadians(getRotation()));   // thrust = 0.1
ay = thrust * Math.sin(Math.toRadians(getRotation()));

Formula for converting a vector represented by x and y properties into magnitude and direction properties.

direction = (int)Math.toDegrees(Math.atan2(ay, ax));
thrust = Math.sqrt(ax*ax + ay*ay);

Notice that the first conversion method uses an Actor method named getRotation. This method returns an Actor's current direction in degrees.
