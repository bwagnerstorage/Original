JCreator Workspace: Inversion

Directions

Method fillArray

Write method fillArray. Given a 2D array fillArray populates the array with random numbers in the range of 0 to 9 inclusive.

Method displayArray

Write method displayArray. Given a 2D array displayArray displays the array on the screen in row/column order.

Method invertHorizontally

Write method invertHorizontally. Given a 2D array invertHorizontally inverts the columns of the array. Inverting means column [0] is swapped with column [array2D[row].length - 1], column [1] is swapped with column [array2D[row].length - 2], and so on.

Method invertVertically

Write method invertVertically. Given a 2D array invertVertically inverts the rows of the array. Inverting means row [0] is swapped with row [array2D.length - 1], row [1] is swapped with row [array2D.length - 2], and so on.
