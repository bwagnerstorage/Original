SumDiagonals

Directions

Write a program that allows a user to input the size of a two dimensional array where size is the number of rows and columns of the array. Create an array of the specified size and fill it with random numbers in the range of 0 to 9. Write code that sums the diagonals of the array. Output the array along with the two sums like the sample output shown below.

Source File

SumDiagonals.java

Sample Run 1

Enter 2DArray size -->3
4 4 1
2 4 4
1 5 9

Sum of Diagonals
----------------
Sum1 = 17
Sum2 = 6
