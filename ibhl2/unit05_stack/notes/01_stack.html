Stack

A stack is a LIFO (Last-In-First-Out) data structure. Data enters and leaves the stack from the same location - the top.

Stack Implementation

A stack can be implemented as either a static data structure(array) or a dynamic data structure(object reference). A stack can be implemented fairly easily using an array the only drawback being the fixed size. This, however, can be remedied by using a dynamic array like is done in the ArrayList class.

The dynamic data structure approach requires the use of another class to create the nodes, but it does not have the size limitation of the array approach.

public class ListNode
{
   // Instance Variables
   private Object value;
   private ListNode next;

   // Constructor
   public ListNode(Object initValue, ListNode initNext)
   {
      value = initValue;
      next = initNext;
   }

   // Accessor Methods
   public Object getValue()
   {
      return value;
   }

   public ListNode getNext()
   {
      return next;
   }

   // Mutator Methods
   public void setValue(Object theNewValue)
   {
      value = theNewValue;
   }

   public void setNext(ListNode theNewNext)
   {
      next = theNewNext;
   }
}
