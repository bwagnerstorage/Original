WordSearch

NetBeans Workspace: WordSearch

Directions

Open the NetBeans project provided and complete two methods of the WordSearch class named searchPuzzle and findWord. Your implementation needs to include a recursive solution.

Method searchPuzzle

This method, given a String parameter, searches a 2D array of characters looking for series of letters that match the given word. If a match is found the method returns a value of true; otherwise it returns false. This method should utilize the method findWord in its implementation.

Method findWord

This method recursively searches the 2D array looking for letters that are organized either horizontally, vertically, or match the the letters of keyword. The letters can be layed out in regular order or reverse order. The method has four parameters. The first parameter, word, contains the string for which the method is searching. The second and third parameters contain the row and column of the 2D array where the method begins its search. The last parameter, index, contains the index of the letter in word that the method is currently searching. Utilize method inBounds in your implementation.
