Integer to Binary Converter

NetBeans Workspace: IntegerToBinary

Directions

Open the NetBeans project provided and complete the method convert that converts an integer value into its binary representation. Your implementation needs to include a recursive solution.

Method convert

This method converts the parameter num into it binary representation. The method returns the binary number as a string of zeros and ones.

View the illustration below that demonstrates two techniques used to convert an integer number to its binary equivalent. Using one of these techniques, create a recursive algorithm that will solve the problem.

 

 

To convert an integer value into a string use the Integer class's toString method.

String ans = Integer.toString(14);   // ans = "14"

or you can use the empty string operator("") and the concatenation operator("+").

String ans = "" + 14;   // ans = "14"
