Directions

In this assignment you are going to implement your own linked list collection class named MyLinkedList. This class will have the same functionality as your MyArrayList class because it implements the same interface.

public interface MyList<E>
{
	// appends an item to the end of a list
	public void add(E item);

	// inserts an item into a list at position index
	public void add(int index, E item);

	// removes an item from a list at position index
	public void remove(int index);

	// retrieves an item from a list at position index
	public E get(int index);

	// updates an item in a list at position index
	public void set(int index, E item);

	// returns the logical size of a list
	public int size();
}

Your MyLinkedList class will also incorporate generics.

get Method

The value returned in your get method will need to be cast to an E.

return (E)cur.getValue();    // cast needed

Demo class

Since your MyLinkedList class uses the same interface as your MyArrayList class you can test this class with the same code.
