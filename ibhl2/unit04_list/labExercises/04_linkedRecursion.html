Directions

Because of the way linked structures are organized recursion presents a viable and dare I say elegant way to traverse a set of nodes. Below is a recursive method that prints the contents of a linked list from beginning to end.

public void printList(ListNode cur)
{
    if(cur != null)  // base case
    {
        System.out.print(cur.getValue()+ " ");
        printList(cur.getNext());  // recursive call
    }
    else
        System.out.println();
}

In this assignment you are going to complete two methods: recursiveSearch and numOccurrences. Both of these methods need to be implemented using recursion. The recursiveSearch method should search a linked list for a key value. If the value is found within the list the method it should return true; otherwise it should return false. The numOccurrences method should count and return the number of times a key occurs within a list.

For example, suppose a list contains the following values:

4 8 2 4 5 3 2 4 1 3 9 2

The table below represents several calls to the numOccurrences method.

call to numOccurrences result

numOccurrences(head, 4) 3
numOccurrences(head, 7) 0
