Directions

In this assignment you are going to complete 5 methods named removeFirstNode, removeLastNode, addFirstNode, addLastNode, and insertNode. Each method is passed a reference to a ListNode object in their parameter list named head. After the linked list is modified in the each method the variable head is returned so that the changes are preserved. The creation and display of the linked list is done for you in the main method.

Method removeFirstNode

Removes the first node in head and then returns head. (Note: When I say "head", I actually mean the "linked list referenced by variable head").

Method removeLastNode

Removes the last node in head and then returns head.

Method addFirstNode

Creates a new ListNode object and assigns it the value of num. Adds this new node at the beginning of head and assigns the new node's reference to head which makes the new node the head of the list.

Method addLastNode

Creates a new ListNode object and assigns it the value of num. Appends this new node to the end of head.

Method insertNode

Creates a new ListNode object and assigns it the value of num. Inserts this new node at the nth position in the linked list. For example, if head contained the following nodes

head --> 2 --> 4 --> 6 --> 8 --> null

and insertNode was called with the following arguments insertNode(10, null), then the new node would be inserted into head between 4 and 6.

head --> 2 --> 4 --> 10 --> 6 --> 8 --> null
