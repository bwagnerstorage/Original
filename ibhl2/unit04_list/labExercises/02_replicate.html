JCreator Workspace: Replicate

Directions

Complete the method replicate. Method replicate adds new nodes to a linked list so that the current nodes in list are replicated the number of times specified by parameter n. For example, if the linked list is represented by

      ("red","white","blue")

Then the call replicate(2) should change the linked list as shown below.

      ("red","red","red","white","white","white","blue","blue","blue")

Each node in the list is replicated 2 times.

If the linked list is represented by

      ("red","white","blue")

Then the call replicate(3) should change the linked list as shown below.

      ("red","red","red", "red","white","white","white","white","blue","blue","blue","blue")

Each node in the list is replicated 3 times.
