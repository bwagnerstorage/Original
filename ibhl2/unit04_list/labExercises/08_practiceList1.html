Directions

In this assignment you are going to complete 4 methods named createList, printList, sumList, and filterList. Each method has access to the instance variable head that will contain a reference to the linked list you create in the createList method.

Method createList

Create a linked list of 10 ListNode objects where each node contains a random number in the range of 1 to 50 inclusive. The linked list's reference(address) is assigned to the instance variable head.

Method printList

Displays the linked list reference by the variable head. The list of numbers are displayed horizontally with a space between each one.

Method sumList

returns the sum of the numbers in the linked list referenced by head.

Method filterList

Displays the numbers in head that are greater than or equal to 25.
