Directions

The Java Collection Framework(JCF) provides two linear collection classes named ArrayList and LinkedList. The ArrayList class is implemented as a static data structure using an array and the LinkedList class is implemented as a dynamic data structure using object references. In this assignment you are going to complete the implementation of your own ArrayList class named MyArrayList. Your class will also be implemented as a static data structure with the functionality specified in the interface MyList shown below.

public interface MyList<E>
{
	// appends an item to the end of a list
	public void add(E item);

	// inserts an item into a list at position index
	public void add(int index, E item);

	// removes an item from a list at position index
	public void remove(int index);

	// retrieves an item from a list at position index
	public E get(int index);

	// updates an item in a list at position index
	public void set(int index, E item);

	// returns the logical size of a list
	public int size();
}

Method resize

The following resize method has been implemented in your MyArrayList class.

public void resize()
{
    E[] array2 = (E[])new Object[array.length + 5];  // increase length by 5

    for(int i = 0; i < array.length; i++)
    {
        array2[i] = array[i];
    }

    array = array2; // replace original array with new array
}

This method creates a second array 5 elements larger than the array that stores the data for the class. It then transfers the data from the original array to this new array and throws the old array away. This is the same technique used by the Arraylist class to provide the capability of unlimited size. You will call this method in your add methods when your class's internal array needs to be resized.


Pseudocode for add Methods

The first add method has a fairly straight forward implementation. It checks to make sure the array is not full. If it is not, it appends the new data to the end of the array. If the array is full it resizes the array then appends the data to the end of the array. The last step it performs is to update the logical size of the array.

The second array performs the same size check as the first method. It then pushes all the data in the array down one unit starting at the insertion point to make room for the new data. It finishes by inserting the new data into the array at the insertion point and updates the array's logical size.

Generic Programming

Generics is a Java programming language feature that allows a programmer to design a class where the user of the class is able to specify the type of data that it will contain. For example, a user of the ArrayList class instantiates an object using the following format:

ArrayList 
<Integer>
 list = new ArrayList();

By using the notation <Integer> the user is able to specify the type of data the ArrayList will contain. This is generics.

Your MyArrayList class also incorporates generics. The generic code has already been added to the MyList interface and MyArrayList class so no other generic code is necessary. The E in the code means any object.
