Directions

In this assigmment you are going to complete two methods: hasDuplicates and removeDuplicates. The hasDuplicates method should return true if the linked list, specified in the parameter list, has duplicate items; otherwise it should return false. The removeDuplicates method should remove all duplicate items from the linked list specified in the parameter list.

For example, suppose a list contains the following values:

     4 8 2 4 5 3 2 4 1 3 9 2

A call to hasDuplicates would return true.

A call to removeDuplicates would change the list to:

     4 8 2 5 3 1 9
