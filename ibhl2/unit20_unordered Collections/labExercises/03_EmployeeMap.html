EmployeeMap

Description

A map is a collection that encapsulates an association between objects. Maps use key/value pairs to associate an object (the value) with a key; the map can be sorted or unsorted. Use a TreeMap if you wish the map to be sorted, use a HashMap if sorting is not a priority.

* Note that in a TreeMap, the ordering applies only to the keys, not the values themselves.

Write a program that allows a user to enter employee information into a map and then is able to search the map for an employee using the employee's social security number as the key. The program should also be able to display a list of employees currently in the map. The ordering of the list of employees need not be in any particular order.

Directions

1. Copy the following class into a java file named Employee.java.

public class Employee
{
    // Instance Variables
    private String name;
    private String address;
    private String SSN;
    
    // Constructors
    public Employee()
    {
        name = "";
        address = "";
        SSN = "";
    }
    
    public Employee(String n, String a, String ssn)
    {
        name = n;
        address = a;
        SSN = ssn;
    }

    // Accessor Methods
    public String getName()
    {
        return name;
    }
    
    public String getAddress()
    {
        return address;
    }
    
    public String getSSN()
    {
        return SSN;
    }

    // Mutator Methods
    public void setName(String n)
    {
        name = n;
    }
    
    public void setAddress(String a)
    {
        address = a;
    }
    
    public void setSSN(String s)
    {
        SSN = s;
    }
    
   // toString method
    public String toString()
    {
        String str;
        str = "Name    = " + name + "\n" +
              "Address = " + address + "\n" +
              "SSN     = " + SSN;
        return str;
    }
    
}

printList Method

How do you print the values contained in a map?

1. Get a list (set) of the keys from the map.
2. Use this list (set) of keys to retrieve the values from the map.

The Map method keySet returns the list of keys in a map as a set. If the map that calls the method is a HashMa the set will be returned as a HashSet and unsorted and if the map is a TreeMap the set will be returned as a TreeSet and sorted (by key).

Iterator iterator = map.keySet().iterator;

Note: You can't get an iterator on a map, but you can get an iterator on a set.
