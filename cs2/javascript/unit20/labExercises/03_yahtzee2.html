Player Interface

Our yahtzee game's player interface has been updated with the addition of two more components. A second button has been added named "Reset Turn" and a div has been placed between the two buttons to display the number of dice rolls a player has left.


In yahtzee a player has a maximum of three dice rolls per turn. After the third roll a player must must choose a category on the scorecard to place his score.

Look at the yahtzee object introduced in the previous lesson. A new field has been added named numRolls. This variable will be used to track the number of dice rolls a player has left in the current turn. numRolls starts with a value of 3 and is decreased each dice roll until it has a value of zero.

function yahtzee() 
{
    this.context = null;  // reference to canvas context
    this.dice = [];       // array representing 5 dice
    this.images = [];     // array to store 6 die images
    this.loadCount = 0;   // count of number of images loaded
    this.rolled = false;  // flag to test if a dice roll is in progress
    this.numRolls = 3;    // track dice rolls left
};

Lab Exercise

  1. Modify the rollDice method so that it incorporates the following behavior:

    • The variable numRolls is decreased by one each dice roll.

    • The value of numRolls is displayed in the div with id "rolls".

    • The dice can only be rolled if the value of numRolls is greater than 0.

    Programming Note: When referring to the field numRolls in the rollDice method preference the variable and method names with the word that instead of this. (For example: that.numRolls--;). The that variable is assigned the object referenced by the this variable at the top of the rollDice method. This was done to get around some of the problems associated with using the this variable.

  2. Implement the method endTurn. Method endTurn should do the following only if the value of numRolls is equal to 0.

    • Change each die objects hold field to true.

    • Change the background color of all 5 of the hold buttons to "gold".

    • Disable all 5 of the hold buttons and the "Roll Dice" button. A method named disableButtons has been provided for this purpose.

    Programming Note: The endTurn method is passed a reference to the yahtzee object stored in the parameter obj. When referencing any of the yahtzee fields or methods use this variable. (For example: obj.numRolls).

    Programming Note: A call to the endTurn method has already been added to the rollDice method. It is called after the completion of each dice roll.

  3. Implement the method resetTurn. The resetTurn method is called when the button "Rest Turn" has been selected. Method resetTurn should do the following:

    • Reset the numRolls variable back to 3.

    • Change each die objects hold field to false.

    • Change the background color of all 5 of the hold buttons to "lightgoldenrodyellow".

    • Enable all 5 of the hold buttons and the "Roll Dice" button. A method named enableButtons has been provided for this purpose.

    • Make a call to the method hideDice. This method clears the dice images from the dice panel.
