  1. File Name: 12_1_game100.html

  2. Game100 is a two player numbers game. The game is played by starting a number at 0. Each player takes a turn adding a value between 1 and 10 to the number. The object of the game is to be the player that makes the number equal 100.

  3. The Game100 has the following rules:

    • The game's number starts with a value of 0.

    • Player 1 starts the game.

    • During a player's turn he/she selects a number between 1 and 10 inclusive which increases the game number by that value.

    • Players alternate turns.

    • The player who selects a number that when added to the game number equals 100 is declared the winner.
