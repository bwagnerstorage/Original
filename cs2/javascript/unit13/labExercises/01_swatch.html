  2. An event handler is a method that is called to respond to an event such as a key press, mouse movement, or button click.

    So far you have declared event handlers with code like the following:

    <button type="button" onClick="split()">Split</button>

    The event handler in this code is the function split(). This function is called in response to the designated button being clicked.

  3. Another way to declare an event handler is to use Javascript to dynamically assign a function as an event handler to a particular event. Look at the following example: 
    div.onmouseover = function(event)
{
    document.getElementById("output").style.backgroundColor = "blue";
};

    The Javascript above assigns an anonymous function as an event handler for a div's onmouseover event. An anonymous function is a function defined without a name. Since the function is being assigned as the event handler for the onmouseover event as it is being created a name is not necessary. Any code placed inside this function will be executed anytime the mouse hovers over the top of the designated div. The parameter event is an object variable that contains information about the event that occurred. For example, the statement: event.target gives the event handler access to the object that was effected by the mouseover event.

  4. Another cool feature of Javascript is that it can add new elements to a document's html. Look at the following example:

    var newDiv = document.createElement("div");
newDiv.setAttribute("class", "div1");              
newDiv.setAttribute("style", "background-color:rgb(0,144,192)");
var text = document.createTextNode("Happy Days");         
document.getElementById("output").appendChild(newDiv);    
document.getElementById("output").appendChild(text);

    The "output" div before the code above was executed:

    <div id="output"></div>

    The "output" div after the code above was executed:

    <div id="output"><div class="div1" style="background-color:rgb(0,144,192)">Happy Days</div></div>

This function uses the three global variables red, green, and blue. Each time the function is called it returns a string representation of a unique background color attribute.

  7. The palette should include the following features:

    • each square is a div element dynamically created by Javascript

    • each div's class attribute is set to "box"

    • each div's background attribute is set to a color provided by the function makeBkColor()

    • each div is appended to the "swatch" div's innerHTML attribute
