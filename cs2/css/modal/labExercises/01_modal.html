Lab Exercise

  1. Open VS Code and create a new document named 28_1_modal.html and save it to your CSS folder.

  2. Give the document the title: 28_1_modal.

  3. In VS Code create a file named 28_1_modal.css and save it to your CSS folder.

  4. Use external styling to create the document shown below.

  5. A modal dialog is a window that forces the user to interact with it before they can go back to using the parent application. A good example of this would be a prompt for saving, or the "open file" dialog. They are often used when a user is forced to make an important decision.

  6. Create a document that displays a modal window like the one show below.

  7. Click on the following link: Bronco Background Image and save the image to your css folder.

  8. Create a div element with the id background.

  9. Use the following styling with the div:

    #background {
    background-image: url("Bronco.png");
    background-repeat: no-repeat;
    background-attachment: fixed;
    background-position: center;
    background-size:contain;
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    opacity: 0.2;
    z-index: -1;
}

    This styling will make the Bronco image the background for the document. Notice I did not set the background image by styling the body element. The reason I used a div is because you cannot change the opacity of a background assigned to the body element.

  10. Create an anchor tag(hyperlink) below the background div that will display a modal dialog when selected. To do this use the following code:

    <a href="#showModal">Display Modal</a>

  11. To create a modal follow these steps:

    • Create a div below the anchor tag and assign it the id name showModal and the class name modal.

    • Style the div using its class name so that it covers the entire document window.

    • Have the modal div start invisible by setting its display property to none.

  12. Change the modal div's property attributes when the user clicks on the link created in step 10 by following these steps:

    • Add the following code to your styling:

      .modal:target {

}

      The div's target selector is activated when the user clicks on the link that references the div.

    • Inside the target selector set the display property to block so that the modal div will become visible when the link is selected.

    • Change the modal's background color transparency so that it will look like it is graying out the document window.

  13. Create another div element inside the modal div and assign it the class name modalDialog.This div will be a small dialog window that will appear on top of the modal div.

  14. Animate the modalDialog div so that it appears above the top of the modal div's window and moves down until it reaches the center of the div. To accomplish this add the following styling:

    .modalDialog {
    position: relative;
    width: 400px;
    height: 200px;
    top: 200px;
    margin: auto;
    padding: 10px;
    border: 1px solid black;
    border-radius: 10px;
    box-shadow: 4px 4px 6px #000;

    background: -moz-linear-gradient(#1A001A, #DA70D6);
	background: -webkit-linear-gradient(#1A001A, #DA70D6);
	background: -o-linear-gradient(#1A001A, #DA70D6);
    background: linear-gradient(#1A001A, #DA70D6);

        /* Add animation */
   -webkit-animation-name: movedown; /* Chrome, Safari, Opera */
   -webkit-animation-duration: 0.9s; /* Chrome, Safari, Opera */
   -webkit-animation-fill-mode: forwards; /* Chrome, Safari, Opera */

    animation-name: movedown;
    animation-duration: 1s;
    animation-fill-mode: forwards;
}

    15. Add an anchor tag to the modalDialog div and assign it the class name close. Place an X as the text for the anchor tag.

    Add the following styling: 

    .close {
    background: #000000;
    color: #ffffff;
    line-height: 32px;
    position: absolute;
    font-family: "Consolas", sans-serif;
    right: -12px;
    text-align: center;
    top: -10px;
    width: 32px;
    text-decoration: none;
    font-weight: bold;
	
    -webkit-border-radius: 15px;
    -moz-border-radius: 15px;
    border-radius: 15px;
	
    -moz-box-shadow: 2px 2px 4px #000;
    -webkit-box-shadow: 2px 2px 4px #000;
    box-shadow: 2px 2px 4px #000;
}

.close:hover { background: #FFCC00; }

  15. Style the modalDialog div however you like. I used a purple gradient for the background color. For the text I used text-shadowing.
