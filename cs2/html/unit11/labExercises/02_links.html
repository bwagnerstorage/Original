11-2 HTML Target Attribute

Chapters

  • HTML Links

Lab Exercise

  1. Open Notepad and create a new document named 11_2_target.html and save it to your HTML folder.

  2. Give the document the title: 11_2_target.

  3. Create a heading labelled CompSci Website.

  4. Create a link labelled CompSci that when clicked will open the webpage http://bwagner.org.

  5. Assign the anchor tag's target attribute a value of _blank so that the link when clicked will open bwagner.org in a new browser tab.

  6. Use internal styling(css) to modify the links link, visited, hover, and active default colors.
