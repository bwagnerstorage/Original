Lesson

An editor is a computer program that enables a user to add or edit text. Editor's can be as simple as NotePad or as complex as Microsoft Word. In the world of web design, there are editors specifically designed to make the creation of web documents easier and faster. Here is a list of some of the most popular of today's editors.

  • NotePad++

  • Sublime Text

  • Atom

  • Vim

  • Visual Studio Code

  • Brackets

All of the editors listed are free to use except for Sublime Text. In this course, we will be using Visual Studio Code developed by Microsoft.


Visual Studio Code Installation using Chrome

  1. Click on the following link: Visual Studio Code

  2. If the download does not start automatically, then click the green Download for Windows button.

  3. In the Downloads window, when the file VSCodeUserSetup is completely downloaded, that is, when the icon no longer has a blue circle around it.

    4. Then click on it once to start the installation process.

  4. The next window, select "I agree with the statement" and click Next button.

  5. The next window, place a check mark on all boxes, then click the Next button.

  6. The next window, click the install button to complete the installation.

  7. When the installation is complete there should be a VS Code shortcut icon on the desktop. Drag the icon onto the Taskbar, then click the icon to launch the program.

  8. Spend some time exploring the program. Below is a link to a list of documents and videos that demonstrate some of its features.
