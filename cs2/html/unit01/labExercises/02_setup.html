1-2 Setup Learning Environment

Lesson

In this lesson you are going to add two folders to the computer's desktop that will contain all of the files you will be creating as you complete the lessons. You are also going to sync these folders with your DentonISD Google account so that your folders will automatically be backed up to your GoogleDrive.


Add Working Folders

  1. Right-click on your computer's desktop. From the menu, select New > Folder.


  2. Give this new folder the name MyCS2.


  3. Double-click on the MyCS2 folder to open it. Create a folder in this window and assign it the name MyHTML.



Google Drive File Stream

The Google Drive File Stream app is a program that can be installed on a computer to provide users access to their Google Drive accounts without the need of a browser. The app assigns a drive letter to users Google Drive so they can use a computer's file system (i.e Window's Explorer) to access the files and folders in their Google Drive.

To begin using Google Drive File Stream let's first determine if the app is installed on your computer. Click on the Start button and scroll the list of apps until you reach the letter D.

You should see an icon named Drive File Stream. If it is not there you will need to follow the steps in the section below entitled Install Google Drive File Stream Application.

If the icon is on the app list, then skip down to the section Sign-in to Drive File Stream.


Install Google Drive File Stream Application

  1. Click on the Start Button icon in the lower left corner of the screen.

  2. Select the Software Center app located on the Start menu.

  3. When the Software Center opens give it about 20 seconds to allow the program to populate the window with all available applications, then browse the list to locate the program Google Drive File Stream.

  4. Double-click the icon to install the app.



Sign-in to Drive File Stream

  1. If you just installed the software the following sign-in window should appear. If the program was already installed, then click on the Drive File Stream icon in the Start menu's app list to make the sign-in window appear.

    2. Enter your school google account email address (my_id@g.dentonisd.org).

  2. Next window, enter your password.

  3. Next window, click the right arrow button twice.

  4. Click the button to open the Google Drive File Stream window.

  5. Next window, double-click on My Drive

    .

  6. Leave this window open for the next section.



Move your MyCS2 Folder to Your School GoogleDrive Account

  1. If not open, then open the Google Drive File Stream window.

    Minimize or reduce the size of the Google Drive File Stream window so that you can see the computer's desktop.

  2. Copy and Paste or Click and Drag the MyCS2 folder and place it in the My Drive window.

  3. Right-click on the MyCS2 folder in your My Drive and select New > Shortcut from the menu.

  4. Copy and Paste or Click and Drag the MyCS2 folder shortcut and place it on the computer's desktop.

  5. Your MyCS2 folder in your My Drive is now accessible through the shortcut you placed on your desktop.


Click the yellow Folder icon located on the Taskbar. Click the This PC icon located in the directory tree. In the This PC window notice that your Google Drive has been assigned a drive letter (In my case, it called the I: drive). This means you now have access to your google drive through Windows Explorer instead of having to view it in Google Chrome.
