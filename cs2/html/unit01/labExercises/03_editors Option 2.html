Lesson

An editor is a computer program that enables a user to add or edit text. Editor's can be as simple as NotePad or as complex as Microsoft Word. In the world of web design, there are editors specifically designed to make the creation of web documents easier and faster. Here is a list of some of the most popular of today's editors.

  • NotePad++

  • Sublime Text

  • Atom

  • Vim

  • Visual Studio Code

  • Brackets

All of the editors listed are free to use except for Sublime Text. In this course, we will be using Visual Studio Code developed by Microsoft.


Note: The following instructions are specific to computers in our lab. Typically, when Visual Studio Code is installed on a computer it is not placed in the folder shown below.

Visual Studio Code Setup

Visual Studio Code is already installed on your computer you just need to create a shortcut for it and place it on the desktop so that it is easy to access. To create a shortcut do the following:

  1. Right-click on the Windows Start Men Icon in the lower left corner of the screen and select the Run option from the menu.

  2. In the Run Window's textbox type the following directory path: c:\Users\Public\Microsoft VS Code v1.37


  3. In the next window, right-click on the file Code.exe to select it, then select the Create shortcut option from the menu.

  4. Finally, click and drag the shortcut file to the computer's desktop.

  5. Optional: change the name of the shortcut to "VS Code". Drag the shortcut to the taskbar.

  6. You can now use the shortcut to open VS Studio Code.
