Selecting a Color

There are three ways you can specify a color using the style's attribute.
By the color's name, hex value, or RGB value.

Color Name

Click on the link: Color Name

<style="color:crimson">

Hex Value

Scroll through the list of color names until you find crimson. Out beside the name is crimson's hex value: #CD143C.
These are base 16(hex) numbers.

<style="color:#CD1430">

RGB Value

To see crimson's RGB value click on either its name or hex value. The RGB value is 220,20,60.
These are base 10(decimal) numbers.

<style="color:RGB(220,20,60)">

  3. Copy the following starter code into your html document.

<html>
<head>
   <title>6_1_styles</title>
