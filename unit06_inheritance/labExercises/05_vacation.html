Directions

Wally World Resorts has asked you to design a vacation planner for their customers. The resort offers three vacation packages: the Standard Package, the Premium Package, and the Platinum Package. Each package offers vacationers a different resort to meet their needs and budget. With the Standard Package guests stay at the Wilderness Resort at a cost of $100.00 a night. With the Premium Package guests stay at the Carribean Resort at a cost of $150.00 a night. With the Platinum Package guests stay at the Spa Spectular Resort at a cost of $200.00 a night. The Premium Package offers guests a meal plan that provides guests with breakfast at no additional cost. The Platinum Package offers guests a meal plan that provides guests with breakfast, lunch, and dinner at no additional cost.

The StandardPackage class is defined with the following attributes and behaviors. Copy the following code into a source file named StandardPackage.java.

public class StandardPackage
{
    // Instance Variables
    private int nights; 
    
    public StandardPackage(int n)
    {
        nights = n;
    }
    
    public String getResort()
    {
       return "Wilderness Resort";  
    }
    
    public double getCostPerNight()
    {
       return 100.00;
    }
    
    public double getTotalCost()
    {
        return nights * getCostPerNight();
    }
    
    public String toString()
    {
        String str = "Resort           = " + getResort() + "\n" +
                     "Cost per night   = " + getCostPerNight() + "\n" +
                     "Number of nights = " + nights + "\n" +
                     "Total cost       = " + getTotalCost();
        return str;
    }
}

Modifications

    Make the following additions and modifications to the PremiumPackage class:

  1. Rule : A subclass can add new private instance variables.
    - none

  2. Rule : A subclass can add new public or private methods.
    - Add a method named getMealPlan which returns the string "Breakfast".

  3. Rule : A subclass can override (redefine) inherited methods.
    - Override the getResort method so that it returns the resort name "Carribean Resort".
    - Override the getCostPerNight method so that it returns a value of $150.00.
    - Override the toString method so that it includes information about the type of meal plan (call the getMealPlan method).

  4. Rule : A subclass must define its own constructors.

To test the functionality of your classes copy the following code into a source file named VacationPlanner.java.

import java.util.*;

public class VacationPlanner
{
    private StandardPackage vacationPackage;
    
    public VacationPlanner()
    {
        Scanner keyboard = new Scanner(System.in);
        int pack=0;
        int nights=0;
        
        do
        {
            System.out.println("Select Vacation Package");
            System.out.println("-----------------------");
            System.out.println("1. Standard Package");
            System.out.println("2. Premium Package");
            System.out.println("3. Platinum Package");
            System.out.println("4. Exit");
            System.out.println();
            System.out.print("Select package --> ");
            pack = keyboard.nextInt();
            
            if(pack < 4)
            {
                System.out.println();
                System.out.println("Enter Vacation Info");
                System.out.println("-------------------");
                System.out.print("Enter number of nights -->");
                nights = keyboard.nextInt();
                System.out.println();   
    
                if(pack == 1)
                {
                    vacationPackage = new StandardPackage(nights);
                }
                if(pack == 2)
                {
                    vacationPackage = new PremiumPackage(nights);
                }
                if(pack == 3)
                {
                    vacationPackage = new PlatinumPackage(nights);           
                }
                
                System.out.println("Vacation Package Summary");
                System.out.println("------------------------");
                System.out.println(vacationPackage);
                System.out.println();
                System.out.println();        
            }           
        }
        while(pack < 4);     
    }
	
    public static void main(String[] args)
    {
        VacationPlanner app = new VacationPlanner();
    }
}
