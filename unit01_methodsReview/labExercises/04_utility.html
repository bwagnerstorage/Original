Directions

The code below is an example of the way we typically retrieve user input from the keyboard

Scanner keyboard = new Scanner(System.in);
System.out.print("Enter a number -->");      // user prompt
int num = keyboard.nextInt();

To demonstrate static methods you are going to define 3 static methods within a class named Utility that will simplify the code above by first eliminating the need to instantiate a Scanner object and second it will combine together the user prompt and the line that reads the input. As an example, the code below also retrieves user input from the keyboard using a static method defined in the Utility class.

int num = Utility.readInt("Enter a number -->");

To display blank lines on the console window you have to include multiple printlns like the code below.

System.out.println();
System.out.println();
System.out.println();

In the Utility class you are going to define a static method that displays blank lines using the following syntax

Utility.blankLines(3);

where the argument 3 indicates how many blank lines to display.

Method Implementation

  1. Implement method readInt. This method prompts a user to enter an integer value from the keyboard. It then reads and returns the integer value using the Scanner class's nextInt method.
    Add a dummy read to the method just before the return line. Example, 
    keyboard.nextLine()

    *If you don't remember what a dummy read is click on the following link: What is a dummy read?

  2. Implement method readDouble. This method prompts a user to enter a double value from the keyboard. It then reads and returns the double value using the Scanner class's nextDouble method.
    Add a dummy read to the method just before the return line.
  3. Implement method readString. This method prompts a user to enter a string from the keyboard. It then reads and returns the string using the Scanner class's nextLine method. This method does not require a dummy read.
  4. Implement method blankLines. This method prints blank lines on the console window.

To test your Utility class copy the following code into a file named UtilityTester.java.

import java.util.*;

public class UtilityTester
{
    public static void main(String[] args)
    {
        int iNum = Utility.readInt("Enter an integer -->");
        System.out.println("The number = " + iNum);

        Utility.blankLines(2);

        double dNum = Utility.readDouble("Enter a double -->");
        System.out.println("The number = " + dNum);

        Utility.blankLines(2);

        String str = Utility.readString("Enter a string -->");
        System.out.println("The string = " + str);

        Utility.blankLines(2);
    }
}
