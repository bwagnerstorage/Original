UIL Computer Science Contest

Directions

The UIL has a Computer Science contest. In this contest contestants take a multiple choice test consisting of 40 questions. Contestants are rewarded 6 points for each correct answer and they are deducted 2 points for each incorrect answer. No points are deducted for skipping a problem. Write a program that calculates contestants scores.

  • Input includes the contestants name, number correct, and number incorrect.
  • Output includes the contestants name, number correct, number incorrect, number of skips, and score.
  • You need to use the Utility class in your implementation.
  • Your program should be decomposed into appropriate methods.

Source File

UILCompSci.java

Sample Run

Enter contestant's name --> John Wayne
Number correct --> 30
Number incorrect --> 6
