As we have seen, the array is a very powerful data structure that has many uses. It, however, has one major drawback. Arrays are assigned a size at creation and this size cannot be changed. Java provides a collection class named ArrayList that is very similar to an array except that it can be sized dynamically while your program is running. Since the array list can grow and shrink while your program runs it is a more efficient use of memory, and can also be simpler to use.

Note: a collection is an object that groups multiple elements into a single unit. Some collections allow duplicate elements and others do not. Some are ordered and others unordered. Java provides several collection classes including ArrayList, LinkedList, Stack, TreeSet, HashSet and Vector.

ArrayList vs Array

Like an array, an ArrayList is an object that contains a sequence of elements that are ordered by position. Elements in both structures can be accessed and replaced at a given position. However, there are several ways in which an ArrayList is unlike an array:

  • The programmer must use methods rather than the subscript operator [] to manipulate elements in an ArrayList.
  • An ArrayList tracks both its logical size (the number of elements that are currently in it) and its physical size (the number of cells available to store elements).
  • When the programmer creates an ArrayList, its logical size is 0. When the programmer inserts elements into an ArrayList, the list automatically updates its logical size and add cells to accommodate new objects if necessary. The list's logical size is also updated when an element is removed.
  • The positions available for access in an ArrayList range from 0 to its logical size minus 1.

Declaring and Instantiating an ArrayList

To declare an ArrayList you use code like the following

ArrayList<String> list = new ArrayList<String>();

Since ArrayList is a class you must use the new operator followed by a constructor call to instantiate an ArrayList object.

Notice the use of the notation <String>. This called generics in Java. Generics is the method used to specify the data type for the list. With the release of Java 7 the second generics declaration became optional, however you still need to include the brackets. 

ArrayList<String> list = new ArrayList<>();
Note: ArrayList is included in the same package as the Scanner class, therefore they use the same import statement:

import java.util.*;

ArrayList Methods

add Methods

ArrayList provides two methods for adding elements to the list. Both of them are named add. This is an example of the use of method overloading which is one form of polymorphism.

The first add method works like an append by adding the element to the end of the list. The method takes one parameter.

list.add("sunshine");

The second add method takes two parameters and works like an insert. The first parameter is represents the location in the list where the element is to be insert. Starting at the given index position all elements below this position are pushed down one.

list.add(3, "sunshine");

get Method

The get method works like the array's index operator [ ]. It takes one parameter which is the location of the element to be retrieved.

String str = list.get(5);

remove Method

The remove method takes one parameter which is the location of the element to be removed. Starting at the given index position all elements below this position are moved up one. The method also returns the deleted item.

String str = list.remove(5);

set Method

The set method replaces an element at a given location. The method takes two parameters. The first parameter is the index of the element to be replaced and the second parameter is the element's new value. The method returns the old element.

String str = list.set(5, "rainbow");

or

list.set(5, "rainbow");

size Method

The size method returns the logical size of the ArrayList. The size method is diiferent from the array's length variable and the String class's length method which return the physical size.

int numElements = list.size();

Method Summary

Here is a list of the most commonly used ArrayList methods.

ArrayList Method Description
ArrayList <E> () Constructor builds an empty array list.
boolean isEmpty() Returns true if the list contains no elements and false otherwise.
int size() Returns the number of elements currently in the list.
E get(int index) Returns the element at index.
E set(int index, E obj) Replaces the element at index with obj and returns the old element.
boolean add(E obj) Appends the specified element to the end of this list. Returns true.
void add(int index, E obj) Inserts obj before the element at index or after the last element if index equals the size of the list.
E remove(int index) Removes and returns the element at index.
int indexOf(E obj) Returns the index of the first instance of obj in the list or -1 if obj is not in the list.

*An ArrayList can store an object of any class. The E in the table above means any class.

Sample Program

// Step 1: import ArrayList

import java.util.*;   // needed for ArrayList

public class ArrayListDemo
{
    public static void main(String[] args)
    {
        
// Step 2: instantiate ArrayList object

        ArrayList<String> list = new ArrayList<>();
        
        
// Step 3: add items to the list

        list.add("Texas");
        list.add("Colorado");
        list.add("Wyoming");
        list.add("Montana");
        
        
// Step 4: retrieve items from the list

        System.out.println(list.get(0));
        System.out.println(list.get(1));
        System.out.println(list.get(2));
        System.out.println(list.get(3));
    }
}
