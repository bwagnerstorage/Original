compareTo Method

When using binary search with an array of objects, we must compare two objects. But objects do not understand the < and > operators, and we have seen that = = is not a wise choice for comparing two objects for equality. However, classes that implement the Comparable interface include the method compareTo, which performs the three different comparisons (= = , < , >). Here again is the signature of compareTo:

public int compareTo(Object other)

The behavior of compareTo is summarized in the table below.

Example Value Returned
obj1.compareTo(obj2)

returns 0, if obj1 is equal to obj2, using equals
obj1.compareTo(obj2)

returns a negative integer, if obj1 is less than obj2
obj1.compareTo(obj2)

returns a positive integer, if obj1 is greater than obj2

For example, the String class implements the Comparable interface; thus, the second output of the following code segment is 0:

String str = "Mary";
System.out.println(str.compareTo("Suzanne"));  // Outputs -6
System.out.println(str.compareTo("Mary"));     // Outputs 0
System.out.println(str.compareTo("Bob"));      // Outputs 11

The other integers are system dependent, but the first should be negative, whereas the third should be positive (in the example given, they are -6 and 11, respectively).
