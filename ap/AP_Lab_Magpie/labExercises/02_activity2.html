Magpie Lab - Activity 2

Introduction to the Magpie Class

To run Magpie follow these steps:

  • Download the NetBean's MagPie workspace provide in the section MagPie Workspace.

  • Open the file MagpieRunner2, compile and execute.

Get to know the Magpie class by typing in the following responses:

  • My mother and I talked last night.

  • I said no!

  • The weather is nice.

  • Do you know my brother?

Exploration

Look at the code. See how the if statement assigns a value to the response and returns that response. The method getRandomResponse picks a response from a group of String objects.

Exercises

Alter the code:

  1. Have it respond "Tell me more about your pets" when the statement contains the word "dog" or "cat." For example, a possible statement and response would be:

    Statement: I like my cat Mittens.

    Response: Tell me more about your pets.

  2. Have it respond favorably when it sees the name of your teacher. Be sure to use appropriate pronouns! If the statement contains the salutation "Mr." then the response should include the "he" pronoun, if the statement contains the salutations "Mrs.", "Ms." or "Miss" the response should include the "she" pronoun, and finally if the statement does not contain a salutation then use "he/she" in the response. For example,

    Statement: Mr. Finkelstein is telling us about robotics.

    Response: He sounds like a good teacher.

  3. Have the code check that the statement has at least one character. You can do this by using the trim method to remove spaces from the beginning and end, and then checking the length of the trimmed string. If there are no characters, the response should tell the user to enter something. For example, a possible statement and response would be:

    Statement:

    Response: Say something, please.

  4. Add two more noncommittal responses to the possible random responses.

  5. Pick three more keywords, such as "no" and "brother" and edit the getResponse method to respond to each of these.

  6. What happens when more than one keyword appears in a string? Consider the string "My mother has a dog but no cat." Explain to your neighbor how you might prioritize responses in the reply method.

Question

  1. What happens when a keyword is included in another word? Consider statements like "I know all the state capitals" and "I like vegetables smothered in cheese." Explain the problem with the responses to these statements.
