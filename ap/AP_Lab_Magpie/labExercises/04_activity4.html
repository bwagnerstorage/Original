Magpie Lab - Activity 4

Introduction

As stated previously, single keywords are interesting, but better chatbots look for groups of words. Statements like "I like cats," "I like math class," and "I like Spain" all have the form "I like something." The response could be "What do you like about something?" This activity will respond to groupings of words.

Prepare

You are going to use the Activity4 project in this activity. You will need to make Activity4 the active project. This can be accomplished by right-clicking on the project name and selecting Sets as Active Project.

Exploration

Get to know the revised Magpie class. Run it, using the instructions provided by your teacher. How does it respond to:

  • I want to build a robot.

  • I want to understand French.

  • Do you like me?

  • You confuse me.

Exercises

Look at the code. See how it handles "I want to" and you/me statements.

Alter the code:

  • Have it respond to "I want something" statements with "Would you really be happy if you had something?" In doing this, you will need to add another else/if statement to the method getResponse. You need to be careful where you place the new check(else/if). See if you can figure out why the order is important?

    You will also need to add a new method to the class named transformIWantStatement. The easiest way to do this is to copy the method transformIWantToStatement and modify the code where necessary.

    For example:

    Statement: I want fried chicken.

    Response: Would you really be happy if you had fried chicken?

  • Now have the program respond to statements of the form "I something you" with the restructuring of the response to the form "Why do you something me?" For example:

    Statement: I like you.

    Response: Why do you like me?

Find an example of when this structure does not work well. How can you improve it?
