The Comparable Interface

The Java standard class library contains a number of important interfaces including Comparable, List, Iterator, and ListIterator, just to name a few. The Comparable interface contains only one method, compareTo, which takes an object as a parameter and returns an integer. This interface is important because it gives us a standard way to compare one object to another.

The compareTo Method

public interface Comparable
{
   int compareTo(Object obj);
}

The compareTo method compares the executing object to the parameter to determine their relative ordering. It returns an integer that is less than, equal to, or greater than zero if the executing object is less than, equal to, or greater than the parameter, respectively.

String Class's compareTo Method

The String class implements the Comparable interface therefore it contains a compareTo method. The String class uses this method to establish lexicographical ordering between two strings.

For example, the code below would assign a negative value to result since the string "cat" comes before "dog" in lexicographical ordering.

int result = "cat".compareTo("dog");

This code would assign a positive value to result since the string "pig" comes after "horse" in lexicographical ordering.

int result = "pig".compareTo("horse");

This code would assign a value of zero to result since the two strings are equal.

int result = "cow".compareTo("cow");
