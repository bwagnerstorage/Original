Quiz

Directions

A set of classes is used to display a quiz in various question formats. The MultipleChoice class is used to create multiple choice questions. The TrueFalse class is used to create true/false questions. To provide a standard functionality between the two classes, both classes implement the Question interface. 

public interface Question
{
    String getQuestion();   // returns the question in the appropriate format
    String getAnswer();     // returns the question's answer
}
