  1. Implement a void method named greeting that includes two formal parameters. The first parameter is of type String named firstName. The second parameter is of type int named num. This method should print the phrase "__________, have a nice day!" num times where the blank is the value of the parameter firstName.
    For example, the call greeting("John", 5) would produce the following output

    5

  3. Implement a non-void method named compare that includes one formal parameter of type String named str. This method should return a boolean value of true if the first character of str is equal to the last character; otherwise, it should return false. For example, the call compare("noon") would return true since both the first letter and the last letter are the same.
  4. Implement a non-void method named reverse that includes one formal parameter of type String named str. This method should return a string that is the reverse of str. For example, the call reverse("computer") would return the string "retupmoc".
  5. Implement a non-void method named numOccurrences that includes two formal parameters both of type String named word and letter respectively. This method should return a count of the number of times letter occurs in word. For example, the call numOccurrences("elephant", "e") would return a value of 2 since the letter "e" occurs in the word "elephant" 2 times.
