StoreLargest

Directions

Write a program that creates two two-dimensional arrays of size 4x4 and fills them with random values in the range of 0 to 9. The program should then compare the corresponding elements of these matrices and place the larger of the two numbers in the corresponding position of a third matrix.

Display all three matrices, like the sample output below, using nested enhanced for loops.
