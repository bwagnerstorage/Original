Declare and Instantiate

Declaring and instantiating two-dimensional arrays is accomplished by extending the processes used for one-dimensional arrays:

int[][] table;            // declare array variable
table = new int[4][5];    // Instantiate array with 4 rows, 5 columns

or

int[][] table = new int[4][5];

Assignment

To assign values to the elements of a two-dimensional array you use two index values representing the row and column. Refer to the following examples.

table[2][3] = 75;       // assign 75 to (row 2, column 3) of the array

matrix[0][0] = "Nero";  // assign "Nero" to (row 0, column 0) of the array

Initializer Lists

Initializer lists can be used with two-dimensional arrays. This requires a list of lists. The number of inner lists determines the number of rows, and the size of each inner list determines the size of the corresponding row.

int[][] table = {{ 0, 1, 2, 3, 4},         // row 0
                 {10, 11, 12,13, 14},      // row 1
                 {20, 21, 22, 23, 24},     // row 2
                 {30, 31, 32, 33, 34}};    // row 3

Two things you need to remember:

  • All the elements of a two-dimensional array must be the same type (int, double, char, etc.)
  • The first index always represents the row the second index represents the column, table[R][C] (To help me remember I think about RC Cola. Does anyone still drink this stuff?)
