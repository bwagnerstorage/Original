Manipulating 2D Arrays

The easiest way to manipulate a two-dimensional array is to use nested for loops. For instance, the following code sums all the numbers in the array table. The outer loop iterates four times and moves down the rows. Each time through the outer loop, the inner loop iterates five times and moves across the columns of the current row.

int sum = 0;

for(int r = 0; r < 4; r++)      // There are four rows r = 0, 1, 2, 3
{ 
   for(int c=0; c < 5; c++)     // There are five columns c = 0, 1, 2, 3, 4
   {
      sum += table[
r
][
c
];
   }
}

This segment of code can be rewritten without using the numbers 4 and 5. The value table.length equals the number of rows, and table[r].length is the number of columns in row r.

int sum = 0;

for(int r = 0; r < 
table.length
; r++)       
{ 
   for(int c=0; c < 
table[r].length
; c++)   
   {
      sum += table[r][c];
   }
}
