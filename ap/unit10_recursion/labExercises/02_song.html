The recursive methods we have looked at so far have all been non-void methods with a return value:

public int factorial(int n)
public int countDown(int n)

However, this is not a requirement when implementing a recursive method. A recursive method can be a void. Look at the code segment below.

public void song(int b)
{
   if( // base case has been reached )
      return;
   else
      song(b-1);   // recursive case
}

One approach used to exit a void method is to use a return statement with nothing after it. Notice that the call to recursive case is made on a line that does not contain a return.

When I was growing up, if we were on a long trip somewhere we would occasionally sing the song "Bottles of Beer on the Wall". This old song is a perfect example of a recursive process. In this assignment you are going to implement a recursive method that writes out the words to this song. Since you are all under the drinking age, we will modify the song a little bit by making the bottles "root beer" instead of "beer". If you don't know the song refer to the Sample Run below.

Copy the following code into a file named SongApp.java

        System.out.println();
    }
}
