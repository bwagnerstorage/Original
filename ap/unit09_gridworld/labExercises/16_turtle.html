Turtle

Directions

A Turtle is a Critter. A Turtle is a Critter in every way except that it only has a lifespan of 10 steps. After 10 steps a Turtle dies and is removed from the grid. Turtles cannot eat TurtleEggs.

A MaleTurtle is a Turtle. A MaleTurtle is a Turtle in every way except that it is always the color blue.

A FemaleTurtle is a Turtle. A FemaleTurtle is a Turtle in every way except that it lays an egg when it comes in contact with a MaleTurtle. The egg is laid in a location, selected at random, from a list of unoccupied locations surrounding the FemaleTurtle. A FemaleTurtle is always green.

A TurtleEgg is an Actor. A TurtleEgg is an Actor in every way except that after being laid it incubates for a period of 5 steps then it becomes a Turtle. The Turtle hatched from the egg has a 50% chance of either being a MaleTurtle or a FemaleTurtle.


Implement the Turtle class. Include an instance variable of type int named lifespan. Initialize lifespan with a value of 10 in a constructor. Remember that Turtles cannot eat TurtleEggs.

Redefine the Turtle class's act method using the code below. Notice the addition of a check to see if the grid is equal to null. This is necessary because Turtles remove themselves from the grid in the processActors method when they reach the end of their lifespan. The methods getMoveLocations, selectMoveLocations, and makeMove can no longer be executed if the Turtle is not associated with a grid.

public void act()
{
     
if (getGrid() == null)
        return;
 
     ArrayList<Actor> actors = getActors();
     processActors(actors);
     if(getGrid() != null)
     {
         ArrayList<Location> moveLocs = getMoveLocations();
         Location loc = selectMoveLocation(moveLocs);
         makeMove(loc);
     }
}

Implement the MaleTurtle class. Initialize its color attribute to blue in a constructor.

Implement the FemaleTurtle class. Initialize its color attribute to green in a constructor.
Include in your implementation the behavior described above.

Implement the TurtleEgg class. Include in your implementation the behavior described above.


Use the TurtleRunner class below to test your new classes.

Copy the following code into a file named TurtleRunner.java.

import info.gridworld.grid.*;
import info.gridworld.actor.*;
import java.awt.Color;

public class TurtleRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();

        FemaleTurtle femaleTurtle = new FemaleTurtle();
        MaleTurtle maleTurtle = new MaleTurtle();

        world.add(new Location(5,5), femaleTurtle);
        world.add(new Location(4,4), maleTurtle);

        world.show();
    }
}
