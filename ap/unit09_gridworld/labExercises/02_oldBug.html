OldBug

Directions

An OldBug is a Bug. An OldBug behaves like a Bug in every way, except that it is too old and too tired to drop any flowers.

Implement the OldBug class then use the OldBugRunner class below to test your new class.

Copy the following code into a file named OldBugRunner.java.

import info.gridworld.actor.*;
import info.gridworld.grid.*;
import java.awt.Color;
import java.util.*;

public class OldBugRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();
		
        world.add(new OldBug());
        world.add(new OldBug());
		
        world.show();
    }
}
