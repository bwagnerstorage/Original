RainbowActor

Directions

A RainbowActor is an Actor. A RainbowActor appears like an Actor in every way, except for its color. Every instance of a RainbowActor object is constructed with a random color.

Copy the following code into a file named RainbowActor.java.

import info.gridworld.actor.*;
import info.gridworld.grid.*;
import java.awt.Color;
import java.util.*;

public class RainbowActor extends Actor
{

}

Implement a default constructor for the RainbowActor class that allows a RainbowActor to be instantiated with a random color.

The following code can be used to create a random color.

int r = (int)(Math.random() * 256);
int g = (int)(Math.random() * 256);
int b = (int)(Math.random() * 256);
Color color = new Color(r, g, b);

To assign a random color to the RainbowActor you will need to use the setColor method inherited from the Actor class.

Copy the following code into a file named RainbowActorRunner.java. This class can be used to test the functionality of your RainbowActor class.

import info.gridworld.actor.*;
import info.gridworld.grid.*;
import java.awt.Color;
import java.util.*;

public class RainbowActorRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();
		
        world.add(new RainbowActor());
        world.add(new RainbowActor());
		
        world.show();
    }
}
