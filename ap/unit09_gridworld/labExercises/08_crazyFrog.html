CrazyFrog

Directions

A CrazyFrog is a Frog. A CrazyFrog behaves like a Frog in every way, except that it when it jumps over a rock it changes its color to a random color. If a CrazyFrog cannot move like a CrazyFrog it should move like a Frog. If a CrazyFrog cannot move like a Frog it should move like a Bug.

Redefine the canMove method so that it will return true if it encounters an empty cell or a flower or a Rock directly in front of its current location. This will ensure that the move method will be called in the act method if the CrazyFrog runs into a rock.

Implement the CrazyFrog class then use the CrazyFrogRunner class below to test your new class.

Copy the following code into a file named CrazyFrogRunner.java.

import info.gridworld.grid.*;
import info.gridworld.actor.*;
import java.awt.Color;

public class CrazyFrogRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();

        CrazyFrog frog = new CrazyFrog();

        world.add(new Location(7,5), frog);
        world.add(new Location(6,5), new Rock());
        world.add(new Location(2,5), new Rock());
        world.add(new Location(0,7), new Rock());

        world.show();
    }
}
