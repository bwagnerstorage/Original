Intro to Python

Learning Python

We will be using the learning site w3schools.com for the majority of our course content. Why use w3schools?

  • It's a free site.

  • It's written with the beginner in mind.

  • With their "Try it Yourself" editor, students can edit Python code and view the result in real time on their site.

Getting Started

  1. Go to the w3schools website. Click on the Tutorial Menu and select Learn Python in the Programming Section.

  2. View the first four lessons in the Python Tutorial: Python Home, Python Intro, Python Get Started, and Python Syntax.

  3. Make sure you test out their "Try it Yourself" editor.

Replit

We will be using the online coding site replit.com as our Python IDE or editor. By using this site your code will be available at home as well as at school. We will look at other Python editors later in the course.

Create a Free Replit Account

  1. Click on the following link to open replit: Start Here!

  2. Click on the blue "Sign up" button in the top right corner of the window.

  3. Use your school id as your UserName, your school email for the Email, and your school password as your Password.

  4. Click Create Account.

  5. If you were successful in creating your account, you should be looking at your home screen. Take a few minutes to familiarize yourself with this screen.

  6. Now see if you can create a new Python repl. (A repl is a page in Replit's online editor). When you create a new repl, Replit proves a name for the repl. Instead of using their made up name give your repl the name MyFirst. The Replit IDE opens a new page with three windows positioned horizontally across the screen. The far left window allows you to view the files that have been created by Replit. The middle window is the editor where you type your code. By default Replit creates a file named main.py for the editor. The window to the right is the output window and it is where you will see your programs output displayed.

  7. Type the following code in the editor window: print("My First Program")

  8. Press the Run button to execute your Python code.

  9. Change the phrase in the print function to "Python is fun!" and Run the program again.
