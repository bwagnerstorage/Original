Lab Exercise

Write a Python function named input_int(prompt) that incorporates the functionality of the program in the previous assignment. The function should prompt the user to enter an integer value. The function should prevent the user from entering any value type other than an integer. The function should return the valid number entered by the user.

Write a program that prompts a user to enter two integer values from the keyboard. The prompt is provided in the function's parameter list. Both of these values need to be verified by the input_int function. Once the two valid integer values have been entered, the two values should be subtract from each other in following
format: quotient = num1 / num2. To insure that the program does not crash with a division by zero error, write a try-except-else block that handles a ZeroDivisionError exception error. If an error does occur the program should display the following message: "Division by zero error. Try Again." and then allow the user to re-enter the two numbers until the division problem does not cause an error. When the division problem does not cause an error display the answer in the following format: num1 / num2 = quotient.
