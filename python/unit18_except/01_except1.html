Lab Exercise

Write a Python program that prevents a user from entering in value that is not an integer. If the user does enter an invalid number, then the program should display the message " - Invalid input. Try again." and allow the user to input a number again. The program should repeat this process until a valid number is input.

Use a try-except-else block in your implementation.
