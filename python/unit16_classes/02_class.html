Python Classes

In this exercise you are going to make a class named Rectangle that can store a rectangle's length and width and calculate it's perimeter and area. You are also going to create a function named main that prompts the user to enter the length and width of a rectangle. It then uses these values to calculate the rectangle's perimeter and area, displaying them on the screen. The main function should allow the user to continue to enter new rectangle information until they choose to stop.

Rectangle Class

Complete the class Rectangle. The class needs to include the following:

  1. A two parameter constructor that initializes a rectangle's length and width variables.

  2. A method named perimeter that uses a rectangle's length and width to calculate and return it's perimeter. The formula for the perimeter of a rectangle is

    perimeter = 2 * length + 2 * width

  3. A method named area that uses a rectangle's length and width to calculate and return it's area. The formula for the area of a rectangle is

    area = length * width

Function main

Complete the function main. The function needs to include the following:

  1. Prompt the user to enter the length and width of a rectangle.

  2. Create a Rectangle object using the length and width provided by the user.

  3. Use the Rectangle object to calculate the perimeter and area of the rectangle and display them on the console window.

  4. Include a while loop that repeats the previous steps until the user decides to stop.
