Python Classes

In this exercise you are going to create two Python classes named Point and Line. The Point class represents a point on a coordinate plane. It stores the x and y values of a single point on the plane. The Line class represents a line on a coordinate plane. It stores two points on the plane that represent the starting point and ending point of a line. The Line class also includes a method to calculate the slope of a line.
