Functions

In this lab, you will be writing four functions that include one or more parameters, but do not have a return value.

Lab Exercise

blankLines Function

Write the function blankLines that includes an integer parameter. The function should display one or more blankLines in the terminal window depending upon the value of the parameter. For example, look at the call made to blankLines below:

print("start")
blankLines(4)
print("stop")

The output would look like the following:

start



stop

There are four blank lines between "start" and "stop".

welcome Function

Write the function welcome that includes two parameters. The first parameter is the name of the person you are welcoming. The second parameter is an integer value that determines how many times the welcome message "Welcome <Name>, have a nice day" appears on the screen. For example, look at the following call made to welcome: welcome("John", 2) would produce the following output.

Welcome John, have a nice day!

randomNums Function

Write the function randomNums that includes three parameters. The first parameter is an integer value that marks the bottom range of a series of random numbers. The second parameter marks the upper range of the numbers. The third parameter is an integer value that represents how many random numbers are generated. Use Python's random module. It has a method named randint: num = random.randint(a, b).

For example, look at the following call made to randomNums: randomNums(5,10,8)
would produce output like the following:

6 8 8 10 5 7 6 9

Eight random numbers in the range of 5 to 10 inclusive.

banner Function

Write a new and improved banner function that takes one parameter, a string value, that represents the message that is displayed within the banner. This message should be centered horizontally within the banner. Include the following two lines of code in your function to help with the centering.

dashLen  = len(dashLine)
