Python Functions

A function is a block of code that is assigned a name. To execute the code in a function use the function's name. This is referred to as a function call. Below is a simply example of a function definition named greeting followed by a call to the function. Rule: a function must be defined before a call can be made to it.

def greeting():
   print("Welcome")

greeting()    # call the function greeting

F-String

Two of the functions you will be writing need there output to be formatted into columns. Refer to the Sample Output below. You can use the String class's format method or a newer short-cut version called F-String released in Python version 3.6. Lab Exercise

In this lab, you are going to write three functions. These functions do not have any parameters nor do they have a return value. Call these functions at the bottom of your source code, below the function definitions.

Banner Function

Write a function named banner that displays a "banner" that consists of the title of the banner surrounded by two dashed lines. The title of the banner is "Computer Science".

convertToBinary Function

Write a function named convertToBinary that displays the decimal numbers from 1 to 15 in one column down the screen and a second column that displays the binary representation of each of these numbers. Use Python's built-in function bin to convert a decimal number to binary.

asciiChart Function

Write a function named asciiChart that displays the ASCII values for the characters from 'A' to 'Z' and 'a' to 'z'. The ordinal values for the characters 'A' to 'Z' are 65 to 90 inclusive and the ordinal values for the characters 'a' to 'z' are 97 to 122 inclusive. Use Python's built-in functions chr and str in your implementation. The function's output should look like the Sample Output below.
