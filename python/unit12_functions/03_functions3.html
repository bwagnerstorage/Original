Lab Exercise

average Function

Write the function average that includes two parameters that represent two numbers that you would like to average. The return value for the function is the average of the two parameters. Call the function average using the following code:

print()
print("Average = " + str(average(90, 100)))

average Function 2

Write another average function that has one parameter which represents a list of numbers that are to be averaged. The return value for the function is the average of the numbers in the list. Call the function average with the following code:

avg = average([10, 20, 30, 40, 50])
print()
print("Average = " + str(avg))

area Function

Write the function area that has two parameters representing the length and width of a rectangle. The return value for this function is the area of a rectangle with the given length and width. Call the function using the following code:

print()
print("Area = " + str(area(5, 10)))
print("Area = " + str(area(20, 20)))
