Lab Exercise

Write a program that allows a person to maintain two lists. The first list stores the names of places they would like to visit and the second list stores the names of places they have already visited. Here is a sample screen of the two lists.

Places I Would Like to Visit
----------------------------
Rome
London
Nashville
Glacier National Park
Ireland
      
      
Places I Have Visited
-----------------------
Yellowstone
Alaska

  1. Create two empty lists named places and visited. The places list stores the names of the places the person would like to visit. The visited list stores the names of the places the person has already visited.

  2. Create a menu system that allows a user to select a menu option from a list of choices. The menu needs to be in a loop until the user selects the E)xit option. Here is what the menu system should look like

    Menu: A)dd D)isplay E)xit M)ove R)emove:

    The menu is on an input line so the user has the option of choosing an "A","D","E","M" or "R". The user should be able to type the letters in either lowercase or uppercase. If the user inadvertently types the wrong letter display the message "Invalid Entry: try again!" then display the menu again.

  3. If the user chooses "A", the program should allow the user to enter a name to be added to the places list. Use the List class's append method in your implementation. If the name was successfully added to the list, then display the message "<NAME> has been added" where <NAME> represents the name entered by the user.

  4. If the user chooses "D", the program should display both the places list and the visited list like the sample output shown above.

  5. If the user chooses "E", the program should exit the loop and end the program.

  6. If the user chooses "M", the program should allow the user to enter a name from the places list and move it into the visited list. If successful, the program should display the message "<NAME> has been moved" where <NAME> represent a name in the places list. If, however, the name was not in the places list, for example, because the user misspelled the name, the program should display the message "<NAME> was not moved" where <NAME> represents the name entered by the user. This should be implemented using the if statement and the in operator. Like this 

    if x in places:

    or this

    if x not in places:

  7. If the user chooses "R", the program should allow the user to enter a name from the places list and remove it from the list. If successful, the program should display the message "<NAME> has been removed" where <NAME> represent a name in the places list. If, however, the name was not found in the places list the program should display the message "<NAME> was not found" where <NAME> represents the name entered by the user. Again use the if statement and the in operator in your implementation.
