Data Structures

Data structures, often referred to as collections, are “containers” that organize and group data according to type. The basic data structures in Python include list, set, tuples, and dictionary. Each of these data structures is unique in its own way. Python disguises between them based on their mutability and order.

Mutability

The Mutability of a data structure refers to whether or not the data stored in the data structure can be changed or modified. If a data structure is mutable it means its data can be modified and immutable means the data cannot be modified. For example, Python's String class is immutable, therefore once a string is created its characters cannot be changed.

Order

When we say that a data structure is ordered, it means that its data has a defined order. For example, strings are ordered, because their characters are assigned an index. The first character is index zero [0], the second character is index one [1], and so on.

List

Lists are mutable and ordered. A list, as its name implies, is a list of data (integers, floats, strings, Booleans, or even other lists or more complicated data types). Like characters in strings, elements of a list are indexed starting at 0 using [ ] syntax.

Creating a List

A List is created using square brackets.

numbers = [10, 20, 30, 40, 50]
colors  = ["red", "white", "blue"]
temps   = [98.6, 99.5, 101.8]
emptyList = []
