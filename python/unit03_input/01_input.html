Lab Exercise

Python has a built-in function named input that allows a program to receive data or input from a user. By default, the value returned by the input function is a string. You will learn about casting in the next unit and how it can be used to cast or convert the value returned by the input function to other data types such as int or float.

  • Click on this link to learn how to use the input function.

  • Login to replit.com and create a new Python repl named Input. Complete the following tasks.

  • Write a line of code that uses the input function to display the prompt "What is your favorite color -->".

  • Store the response to the prompt in a variable named myColor.

  • Write a line of code that uses the input function to display the prompt "What is your favorite number-->".

  • Store the response to the prompt in a variable named myNumber.

  • Write a line of code that uses the input function to display the prompt "It is raining (True or False)-->".

  • Store the response to the prompt in a variable named isRaining.
