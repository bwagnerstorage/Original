While Loop

A loop is a programming feature that allows a piece of code to be repeated. There are two types of while loops: a counter-controlled loop and a sentinel-controlled loop. In this lab, you will be practicing the counter-controlled loop.

Counter-Controlled Loop

A counter-controlled loop is a loop that repeats a predetermined number of times. The condition of this type of loop is controlled by a counter variable. A counter variable is a variable that keeps count of the number of times a loop is executed.

Below is sample code that utilizes a counter variable to repeat a print statement 20 times.

cntr = 0    #counter variable
while cntr < 20:
print("Pyhon is fun!")
cntr+=1   #increment counter variable

Programming Note: In Python, a print statement by default adds a new line character to the end of string being displayed. This results in multiple print statements displaying their values on a new line. If you want multiple print statements to display data on the same line you have to add an argument to the end of the print function like the example shown below.

print("Python is fun!", end=" ")

By default, the end argument is set to a new line character "\n". Setting it to a space " " instructs the print function to place a space after it display's it's last value.
