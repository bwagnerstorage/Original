    • Lab Exercise

    Write a program that allows a person to practice their multiplication skills. The program should include the following features:

    • Prompts the user to enter the number of multiplication problems they would like to do.

    • Uses a counter-controlled loop that allows the user to do the number of practice problems desired.

    • Creates a multiplication problem that includes two operands randomly generated in the range of 0 to 10.

      • Displays the problem number in the following format: "Problem x", where x is the current problem number.

    • On the same line it displays an input prompt that includes the multiplication problem with the following format:
      "operand1 * operand2 = "

      Displays the problem number in the following format: "Problem x", where x is the current problem number.

    • Allows the user to input their answer.

    • Checks to see if the user's answer matches the correct answer. If the user's answer matches, then the program displays the message "Correct!". If the user's answer does not match, then it displays the message "Incorrect. Answer = " along with the the correct answer.

    • After answering a question, the program displays how many problems the user has gotten right out of the number of problems attempted.

    • When the program reaches the number of problems specified by the user, the program displays the percentage of how many problems they got right.

    • Next the program displays a prompt asking the user if they would like to "Go Again (y/n)-->". If the user enters "y", then the program starts over. If the user enters "n" then the message "Good bye is displayed and the program ends. This step requires the use of a sentinel-controlled loop.

    • Refer to the Sample Run below.

    Programming Note: This program utilizes both a counter-controlled loop and sentinel-controlled loop. The counter-controlled loop is nested inside of the sentinel-controlled loop.
