Python Modules

In Python, programmers utilize libraries, packages, and modules to organize their code. A library is a collection of related packages, a package is a collection of related modules, and a module is a collection of functions, classes, and variables that can be used in your programs. Each of the three organizational tools is located in a separate file with the extension "py".

For this lab, we are only concerned with modules. An example of a Python module is the random module. To use variables, functions, or classes within the random module it must first be imported into your program. The random module contains an object variable named random that provides you with the ability to generate pseudo-random integer values. The Random class, and therefore the object variable random, provide two methods for creating random integers within a specific range: randrange and randint. Below is the signature for the randint method.

int randint(a, b)

This method returns a random integer N such that a <= N <= b

Look at the following example:

import random

num = random.randint(1,10)
print(num)

This code generates a random integer value between 1 and 10 inclusive.
