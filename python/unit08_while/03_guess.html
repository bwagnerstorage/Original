    Write a program that plays a guessing game using the following rules:

    • The program generates a random number between 1 and 100.

    • The player attempts to guess the number selected by the computer.

    • If the player's guess is less than the number, the message "Too Low, Try Again." is displayed and the user is prompted to guess again.

    • If the player's guess is greater than the number, the message "Too High, Try Again." is displayed and the user is prompted to guess again.

    • If the player guesses the number correctly, the message "You found it in ___ tries." is displayed with the number of times the player had to guess before the number was found.

    • Play continues until the user guesses the computer's number.

      Hint: Declare a boolean variable and assign it a value of False. When the player guesses the random number correctly set the variable to True. For example,
