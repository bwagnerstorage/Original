Lab Exercise

Write a Python program that, given two points on a line, calculates the slope of that line. Use the slope formula shown below in your implementation. The program needs to use the String class's format method to create an output like the one shown in the Sample Run below.

Slope Formula

slope(m) = (y2 - y1) / (x2 - x1)



Tip: You can assign a variable it's data type on an input line.

x1 = int(input("Enter x1 -->"))

This code assigns the variable x1 a data type of int.

Sample Run

Enter x1 -->2
Enter y1 -->4
Enter x2 -->6
Enter y2 -->3

A line with the points (2,4) and (6,3)
has a slope of -0.25.
