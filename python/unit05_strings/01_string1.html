What is a String?

As noted previously, strings are one or more unicode characters grouped or "strung" together. Strings in python are surrounded by either single quotation marks, or double quotation marks. Here are a few examples:

"M" or 'M'
"School" or 'School'
"Have a nice day!" or 'Have a nice day!'
"582" or '582'
"455-3903-2180" or '455-3903-2180'

A String is an Array

Each character in a string is assigned a number called an index. The first character is assigned the index 0, the second character index 1, the third character index 2, and so on. Every character is assigned an index, including spaces.

A string is an array, therefore we can use the square bracket operators [ ] to access individual characters within a string by placing a character's index number within the square brackets, like [0], [2], or [35].

   #    0123456789
   s = "I love you"

   print("s[0] = " + s[0])    # displays "I"
   print("s[1] = " + s[1])    # displays " "
   print("s[2] = " + s[2])    # displays "l"
   print("s[5] = " + s[5])    # displays "e"
   print("s[9] = " + s[9])    # displays "u"

Executing or running the sample code above produces the following output.

s[0] = I
s[1] =  
s[2] = l
s[5] = e
s[9] = u

Looping Through a String

Since strings are arrays, we can iterate through all the characters in a string, with a for loop and the in operator.

The word iterate means to visit every character in a string from its first index 0 to its last index.

#        0123456789
words = "I love you"

for x in words:
   print(x)

Executing or running the sample code above produces the following output.

I

l
o
v
e

y
o
u

The for command is responsible for iterating through a string, copying each character, one at a time, into the variable x. For example, after the first iteration or loop through the string words, x contains the character "I", the second iteration x contains the character " ", the third iteration x contains the character "l", and so on.

The in operator's job is to stop the for from iterating once it reaches the last character in word. It accomplishes this task by verifying the character currently in x is in fact a character in the string. Typically, when the for loop reaches the end of the string it is iterating, the next character iterated is a space or end-of-line marker. If the in operator sees one of these characters it knows the iteration should be terminated, because these characters are not "in" the string word.

We will explore for loops in more detail in a later unit.

String Length

Use Python's built-in function len to get the length of a string. The length of a string is equal to the number of characters in the string, including spaces. Here are so examples of calls to the len() function.

a = "apple"
b = "bananas"
c = "cantaloupe"
d = "Grown on the Farm"

print("Length of 'apple' is " + str(len(a)))
print("Length of 'bananas' is " + str(len(b)))
print("Length of 'cantaloupe' is " + str(len(c)))
print("Length of 'Grown on the Farm' is " + str(len(d)))

Executing or running the sample code above produces the following output.

Length of 'apple' is 5
Length of 'bananas' is 7
Length of 'cantaloupe' is 10
Length of 'Grown on the Farm' is 17

The length of a string can also be calculated by adding 1 to last index number in a string. For example, an "e" is the last letter in the string "apple". The "e" is at index 4, so the length of the string "apple" is 4 + 1 = 5. *Remember that the indexes of a string start with the index 0, not 1.
