Lab Exercise

Write a Python program that does the following:

  1. Allows the user to input a string value from the keyword with the following prompt: "Enter a string-->"

  2. Use the slice syntax to extract the first character of an input string and store the value in variable named first.

  3. Use the slice syntax to extract the last character of an input string and store the value in variable named last. You will also need to utilize the len function.

  4. Concatenate the variables first and last together and store the combined string in a variable named combined.

  5. Concatenate the variables first and last together in reverse order and store the combined string in a variable named reversed.

  6. Display the variable first on the console window with an appropriate label.

  7. Display the variable last on the console window with an appropriate label.

  8. Display the variable combined on the console window with an appropriate label.
