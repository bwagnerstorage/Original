This lab contains classes used to develop tests and/or quizzes that contain both Multiple Choice and True/False questions. The program uses inheritance to create the two types of questions.

Your assignment is to create two classes named MultipleChoice and TrueFalse that inherit the functionality of a base or parent class named Question. These two classes contain information needed to create multiple choice and true/false questions.

    class Question:
  def __init__(self, type, subject, topic):
      self.type = type
      self.subject = subject
      self.topic = topic

  def info(self):
    str = "Question Type: " + self.type
    if self.subject != None:
      str += "\nQuestion Subject: " + self.subject 
    if self.topic != None:
      str += "\nQuestion Topic:" + self.topic
    return str;

MultipleChoice Class

The MultipleChoice class is a subclass of the Question class. It contains information needed to create a multiple choice question. This includes the question, four possible answer choices, and the correct answer.

The class defines a constructor that initializes the Question class's variables, as well as the variables needed to create a multiple choice question.

Look at the following constructor header for the Multiple Choice class: 

def __init__(self, type, question, A, B, C, D, answer, subject=None, topic=None):

Notice the subject and topic parameters are assigned a value of "None". This syntax is used to make these variables optional. This means when you are creating a Multiple Choice question you have the option of assigning or not assigning a value to these variables making them optional. Any variable you want to declare as optional must be listed at the end of the constructor's parameter list.

Look at the info method declared in the Question class. Because the subject and topic variables are declared as optional in the Multiple Choice and True/False constructors it is necessary to include if statements in the info method that test to see if the variables subject and topic are assigned a value of None. This will prevent a null pointer exception error from occurring when displaying question info.

The class also needs to include a new method named display that returns a string that formats a multiple choice question in the following form:

Question
A. Choice A
B. Choice B
C. Choice C
D. Choice D

TrueFalse Class

The TrueFalse class is a subclass of the Question class. It contains information needed to create a True/False question. This includes the question and the correct answer which can be either True or False.

The class defines a constructor that initializes the Question class's variables, as well as variables necessary to create a True/False question.

The class also needs to include a new method named display that returns a string that formats a true/false question in the following form:

Question [T/F]
