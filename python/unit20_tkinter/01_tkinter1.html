Tkinter

Tkinter is the defacto GUI library for Python. If you want a quick and easy way to write GUI programs in Python then Tkinter is probably your best choice. There are a multitude of resources to help you get started including online tutorials and videos.

Lab Exercise

In the previous unit you made a text-based Tic-Tac-Toe game. In this assignment, your task is to convert this program into a graphical program so that the tic-tac-toe grid is drawn on a GUI window. Using a Model-View Architecture you should be able to use the logic and data structures (2D Array) from your text-based game to simplify the program's design.

At the top of this page I have listed several Tkinter tutorials that might be useful to you, but there are probably better one's out there.
