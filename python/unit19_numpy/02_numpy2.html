NumPy Random Module

NumPy has a sub-module named random that allows you to assign random numbers to a NumPy arrays. The random module has a function named randint that assigns random integers to an array. Look at the following example:

a = np.random.randint(low=0, high=100, size=(rows, cols))

The first parameter low sets the lower bound of the generated random number. The second parameter high sets the (upper bound - 1) of the generated random number. The lower bound is inclusive and the upper bound is exclusive, therefore the range of random numbers generated are between 0 and 99 inclusive. The third parameter size is a tuple that represents the dimensions of the 2D array being created.

The labels low, high, and size in the randint's parameter list are programmer defined but also optional. Look at the same code again, but with the labels removed: 

a = np.random.randint(0, 100, (rows, cols))

Personally, I like the labels.
