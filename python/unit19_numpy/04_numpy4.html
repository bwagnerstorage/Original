Lab Exercise

Write a program that allows to people to play Tic-Tac-Toe. Your program does not have to have the same look and feel as the one in the Sample Run below, as long as it allows two people or one person and the computer to play the game. You will need to write the program using either functional programming or object-oriented programming, but I will leave that decision up to you.

Programming Note: To create a 2D of Strings using NumPy you need to use code like the following: 

self.board = np.zeros((3,3), dtype="object")
