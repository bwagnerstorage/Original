Lab Exercise

In this lab your going to write a Student Records program that keeps track of students last names, first names, id numbers, and their classification (9, 10, 11,12). The program should use a menu system to navigate the program. The data collected by the program,through keyboard input, should be saved to a file on secondary storage device. The program should retrieve the students information each time the program is run storing it in an dictionary data structure. The program should manage the data in several ways. It should be able to add a new student, view all of the students, and search for a specific student by id number.
