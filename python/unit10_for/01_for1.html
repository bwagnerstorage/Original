Lab Exercise

  1. Write a for loop that displays the numbers from 1 to 3.

  2. Write a for loop that displays the numbers from 0 to 20.

  3. Write a for loop that displays the strings in the following list:

    farmAnimals = ["dog", "cat", "pig", "hog", "cow"]

  4. Write a for loop that only displays the words in the following list that are of length 5:

    colors = ["red", "green", "blue", "yellow", "white", "purple", "black", "grey, "orange", "brown"]

  5. Write a for loop that displays the numbers in the following list until it encounters a number less than 0, at which point it exits the loop:

    nums = [9, 50, 36, 18, 3, 88, -1, 12, 45, 94, 67, 0]

  6. Write two nested for loops that use the the following two lists to display the multiplication table shown below:

    row = [1, 2, 3, 4, 5, 6, 7, 8, 9]

    col = [1, 2, 3, 4, 5, 6, 7, 8, 9]

    Use this code to align the numbers into right-justified columns:

    temp = "{:>3}"
print(temp.format(x), end="")

    The ":>3" formatting type is telling the print statement to display each number so they occupy three columns right-justified.
    (">" means right-justified, "<" means left-justified, "^" means centered)
