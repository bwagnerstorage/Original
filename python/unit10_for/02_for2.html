Mean, Median, an Mode

Python Resources

  • Python For Loop

Lab Exercise

Write program, that given a list of numbers, will calculate the mean, median, and mode for the numbers. Mean is the average of the numbers in the list. Median is the number that is the middle of the list when the list is sorted in ascending order and has an odd numbered length. If the list has an even numbered length, then the median is the average of the two numbers that share the middle position in the list. Mode is the number or numbers that occur the most often in the list. If all the numbers in the list occur once in the list, then there is no mode value.

The program should give the user the option of generating the list of numbers randomly or through keyboard input. It should also allow the user to specify how many numbers will be in the list and their range of values.

Your program's output should resemble the sample runs below.

Programming Tips

  • Use the List class's sort method in order to calculate the median and mode.

  • Your program should include both the while loop and the for loop in its implementation.

  • The mode is the most difficult of the three calculations. It will definitely challenge you. Use two nested while loops to iterate the list in order to find the maximum number of times that the numbers occur in the list. For example, the following list has 2 numbers that are repeated 3 times, therefore the maximum number of
    times is 3:


    [1, 2, 2, 2, 5, 7, 8, 8, 8, 9]


  • Use another set of nested while loops to iterate through the list again to find and print all of the numbers that occur in the list the maximum number of times, in the case of the example above, the maximum was 3 times, therefore 2 and 8 are the mode values. If there is no mode value, because all the numbers only occur once, your program should display "none" for the mode.
