File Input

This program reads a data file consisting of all the countries of the world and their surface areas. Each line of the file consists of a country's name and its area separated by a comma. The goal of this program is to read the file, extract each country name and area from the data set, so that the data can be sorted by their areas from largest to smallest. The data is displayed in two columns with the areas in the first column and the countries names in the second column.

Follow the steps below to complete the assignment.

Lab Exercise

  1. Create a new replit named FileInput2.

  2. Click on the following link to download the csv file: countries.csv

  3. Click on the ellipse (three vertical dots) icon in the File section. Select the menu option: Upload File.

    In the Dialog box, navigate to the location of the "countries.csv" file. If you did not move it, it should be in your Downloads folder. Select the file. This should add the "countries.csv" file to your replit.

  4. Double-click on the file "countries.csv" to view it. This file is a list of all the countries in the world and their surface area in square miles. Notice that one line of the file consists of the country's name and it's area separated by a comma, hence a csv file.

  5. Open the "countries.csv" file for reading using the following code:

    with open("countries.csv", "r") as file:

  6. Inside the with statement create a list named countriesList with this code:

    countriesList = []

  7. The code below uses a for-in loop to read each line of the file. It then uses the String class's split method to remove the comma separating the country and its capital. The split method will also place the country and the area in a list of size 2. The code then appends the new list to the countriesList creating a list of lists data structure: 

    for line in file:
   pairList = line.split(",")
   print(pairList)    # view each list created
   countriesList.append(pairList)

  8. Add this print statement outside the scope of the for loop. This will allow you to see the data in a list of lists format. 

    print(countriesList)

  9. Now that the data has been read and stored in a list the next step is to sort the list by each country's area.

    countriesList2 = sorted(countriesList, key=lambda x:float(x[1]), reverse=True)

    This sorting code is fairly complex. There are two takeaways I want you to get from this code. First, the sorted function returns a copy of the countriesList sorted in descending order. Second, because each country's name and its area is stored in a list they are indexed, so it is easy to access them. The code x[1] refers to area of the country.

  10. The last step in the program is display the data in a columnar format. Start by adding this code to give each column a title.

    print("   Area          Country") 
print("----------------------------")

  11. Look at an element from the countriesList:

    ['Afghanistan', '652090\n']

    Notice that the area value contains a new line character "\n". This needs to be removed or it will affect the way the data is displayed.

    The following code uses the String class's strip method to remove the new line marker "\n" from the area. It also utilizes the f-String statement to format the data into two columns with the first column left justified using 14 spaces to display the areas in a comma-based format.

    for line in countriesList2:
   area = float(line[1].strip("\n"))
   country = line[0]
   output = f'{area:<14,.0f}{country}'
      print(output)
