What is a Data Type?

A data type is an attribute associated with a piece of data that tells a computer system how to interpret its value. Python has several built-in data types, but we will only be using the following seven types in this course.

Type Catagory Data Type
Text Type str
Numeric Types int, float
Boolean Type bool
Sequence Types list, tuple, range

Python has a built-in function named type that returns a variable's type.

str = "Maddie"
num1 = 99
num2 = 3.14
ans = True

print(type(str))
print(type(num1))
print(type(num2))
print(type(ans))



This sample code uses the type function to produce the following output:




<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>





The output shows that "Maddie" is of type str (string), 99 is of type int (integer), 3.14 is of type float (floating-point), and True is of type bool (boolean).



What is Casting?


Casting means changing the data type of a piece of data from one type to another. 
   There are two types of casting in Python: implicit casting and explicit casting.
   Implicit casting is performed automatically by Python, so we will not be discussing it here. However, explicit
   casting does require programmer intervention through the use of one of Python's built-in functions like str(), int(), and float(). 



Input Function


We learned previously that the value returned by the input function is of type string. 
   Look at the following example.


weight = input("Enter your weight-->")
print(type(weight))



This sample code produces the following output:




Enter your weight -->135
<class 'str'>





Since a person's weight is typically viewed as a number, you might assume that the value 135 
   would be a number. Instead the output from the code shows that
   the weight value is of type 'str'(string).


The input function's default behavior becomes problematic when the return value is used 
   in an arithmetic expression. An arithmetic expression requires that all the values in the calculation be numbers either int or float. 
   This is an example of a programming problem that would benefit from explicit casting.
