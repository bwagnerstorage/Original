Lab Exercise

  • Click on the link in the Python Resources to learn how to create variables in your programs.

  • Login to replit.com and create a new Python repl named Variable.

  • Create a variable named score and assign it the integer value 12.

  • Create a variable named language and assign it the string "Python".

  • Create a variable named cost and assign it the floating-point value 19.99.

  • Create a variable named finished and assign it the boolean value True.

  • Display the value of each variable with a label describing their contents like the Sample output below.

  • If the value of a variable is not a string you will need to use the built-in function str to cast the value in the variable to a string. For example, the following code uses the str function to cast the score variable's integer value 12 to the string "12": 

    print("Score = " + str(score))

    This cast is necessary because the print statement displays a string first namely "Score = ", therefore anything displayed after the "+" operator must also be a string.

Sample Output

Score = 12
Language = Python
Cost = 19.99
Finished = True
