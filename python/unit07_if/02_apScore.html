Lab Exercise

On the 2009 Computer Science AP Exam the range of composite scores was between 0 and 80. The College Board uses a 5 point system to gauge a student's master of the curriculum for the subject being tested. The table below shows how the exam scores for the computer science exam were translated into the 5 point system.
Composite Score AP Score
62 to 80 5
47 to 61 4
37 to 46 3
29 to 36 2
0 to 28 1

Write a Python program that allows a user to enter in their AP composite score, from the keyboard, and then uses this composite score to determine the student's AP score. The program's output should resemble the sample runs below. Use the elif statement in your implementation.

If a student's AP score is a 5 display the recommendation "Recommendation: Extremely well qualified" after the score. If their AP score is a 4 display the recommendation "Recommendation: Very well qualified". If their AP score is a 3 display the recommendation "Recommendation: Qualified". If their AP score is a 2 display the recommendation "Recommendation: Possibly qualified". If their AP score is a 1 display the recommendation "Recommendation: No recommendation"
