Tuples and lists are very similar. The chart below compares their similarities and differences.

To summarize, there are two main differences between tuples and lists. First, once a tuple is created it cannot be modified. Second, when creating a tuple use round brackets () instead of square brackets [].

Why use Tuples?

  • Since lists and tuples are similar, it begs the question "Why do we need tuples? Well, it has to do with the way the two data structures use memory. Since tuples cannot be modified, they can be optimized by the computer so that they require less memory than a list with the same data. Being immutable also gives tuples a slight advantage in access speeds compared to lists.

  • Tuples are useful for representing what other languages often refer to as records. A record is data structure used to store related information, like your student record. A tuple allows you to group related items together and then use the data as a single entity.
