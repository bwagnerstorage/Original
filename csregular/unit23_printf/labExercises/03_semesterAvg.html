Semester Average

Directions

Write a program that allows a teacher to enter in a student's name, three six weeks averages, and a final exam grade. The program should then calculate the semester average using the following percentages:

     Six weeks averages: 80%
     Final exam : 20%

Semester Avg Formula: sum of six weeks avgs times .8 + final exam * .2

The program should use the printf method to print the contents of the variables to the console window using the format shown in the sample run.

Decompose Problem

Decompose the problem into an appropriate number of methods. Define a non-void method that calculates and returns the semester average.

SemesterAvg.java

Enter students name --> Jimmy Stewart
