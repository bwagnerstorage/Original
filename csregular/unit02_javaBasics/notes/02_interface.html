User Interface Style

Before writing a Java program, you must decide what type of user interface your program will use. There are two choices: the graphical user interface (GUI), familiar to all PC users, and the less common terminal I/O interface or console.

GUI - Graphical User Interface programs

GUI programs enhance the users experience by providing menus, buttons, graphics, and navigation using a keyboard, mouse, or touch-pad.

Terminal or Console programs

Terminal programs allow only a text-based interface and navigation using only a keyboard.

In this course, we will first write Terminal-oriented programs then transition to GUI programs later. There are two reasons for beginning with Terminal-oriented programs.

  1. In Java and many other languages, a terminal interface is easier to implement than a GUI.
  2. There are programming situtations that require terminal I/O rather than a GUI, so familiarity with the techniques of terminal-oriented programming is important.
