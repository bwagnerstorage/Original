Syntax

Every spoken language has a general set of rules for how words and sentences should be structured. These rules are collectively known as the language syntax. In computer programming, syntax serves the same purpose, defining how the program should be structed.

A program's source code must have correct syntax in order to compile correctly and be made into a program. In fact, it must have perfect syntax, or the program will fail to compile and produce a syntax error. A syntax error can be as simple as a missing curly brace or a forgotten semicolon at the end of a statement. Even these small errors will keep the source code from compiling.
