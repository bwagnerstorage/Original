Terminal Output

One of the fundamental features of a computer program is its ability to display information on a computer's screen. Java provides three statements for producing output to the terminal window. They are println, print, and printf. We will only be looking at println in this lesson.

Displaying Strings Using println Statement

In computer jargon a word or phrase is called a string. In Java a string must be enclosed inside a par of double quotes. Here are some examples of strings: "Hello World", "X", " ", "12345", "Have a nice day!!!". To display a string using a println statement you simply place the string inside the parentheses.

public class MyProgram
{
   public static void main(String[] args)
   {
      System.out.println("Hello World");
      System.out.println("X");
      System.out.println("    ");
      System.out.println("12345");
      System.out.println("Have a nice day!!!");
   }
}

The program above would produce the following output.

Hello World
X

12345
Have a nice day!!!

Notice that each string is displayed on a separate line. The println statement appends a newline character to the end of each string so that after the string is displayed the window cursor goes to the beginning of the next line. This causes the next println statement to display its contents on the next line.

Displaying Numbers and Mathematical Calculations Using println Statement

You can also print numbers and mathematical calculations using the println statement. You do not enclose them inside double quotes.

public class MyProgram
{
   public static void main(String[] args)
   {
      System.out.println(5);
      System.out.println(25849);
      System.out.println(98.6);
      System.out.println(5+8);
      System.out.println(20 * 7);
      System.out.println(5 - 2 + 6 * 10);
   }
}

The program above would produce the following output.

5
25849
98.6
13
140
63

Java uses the order of operations(PEMDAS) when performing mathematical calculations.

Displaying Blank Lines

You can display blank lines by using a println statement with nothing in the paretheses.

public class MyProgram
{
   public static void main(String[] args)
   {
      System.out.println("Dog");
      System.out.println();
      System.out.println();
      System.out.println();
      System.out.println("Cat");
   }
}

The program above would produce the following output.

Dog



Cat
