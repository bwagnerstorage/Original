Problem 1

Write a program that prompts a user for his/her name and favorite color then displays the information using the following format: [name] 's favorite color is [color].

An apostrophe and a "s" ('s) should be appended to the person's name.

Sample Run 1

***********
 Problem 1
***********
Enter your name -->Steve
Enter your favorite color -->purple

Steve's favorite color is purple
