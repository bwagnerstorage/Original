Skinny Island

Directions

A Jeroo starts out on Skinny Island on the western shore, without any flowers, and must find a way across to the eastern shore by picking the flowers and using them to break through the nets.

Create a method named disableNet that will pick up a flower directly in front of a Jeroo and toss it onto a net that is on the other side of the flower.

Use the following method declaration

// precondition: a flower is position directly in front of the Jeroo
//               with a net on the other side
// postcondition: the flower has been picked and the net has been disabled
method disableNet()
{

}

Use the following code as your main method.

method main()
{ 
   Jeroo bob = new Jeroo();
   
   bob.hop(3);
   bob.disableNet();
   
   bob.hop(8);
   bob.disableNet();
   
   bob.hop(8);
   bob.disableNet();
   
   bob.hop(4);
}
