In Java strings are objects, not primitive data, therefore they cannot be compared for equality using the "==" operator. Instead to compare two strings to see if they are the same you must use either the equals method or the equalsIgnoreCase method. Look at the following code segment.

   String s1 = "ball";
   String s2 = "Ball";
   System.out.println("Test 1 = " + s1.
equals
(s2));
   System.out.println("Test 2 = " + s1.
equalsIgnoreCase
(s2));

Look at the output for this code:

Test 1 = false
Test 2 = true

The first println statement uses the equals method to compare the string s1 with the string s2. Even though both variables contain the string "ball" the equals method returns a value of false because the variable s2 contains a string that begins with a capital letter. The equals method returns true only if the two strings represent the same sequence of characters.

The second println statement uses the equalsIgnoreCase method. Like its name implies the equalsIgnoreCase method compares the string s1 to the string s2, ignoring case considerations. Two strings are considered equal ignoring case if they are of the same length and corresponding characters in the two strings are equal ignoring case.
