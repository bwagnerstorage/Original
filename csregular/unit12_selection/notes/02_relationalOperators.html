Equality Operator (==)

The == operator is initially confusing to most newcomers to Java. In Java, the symbol = already has a meaning, namely assignment. The == operator denotes equality testing. Look at the example below. 

if(age == 65)         
// if age is equal to 65

{
   System.out.println("I can retire");
}

In the example, the age variable is compared to 65 to see if they are equal.

You will have to remember to use == for equality testing, and to use = for assignment.

Operator Precedence

Below is the entire list of operators available in Java and their precedence levels. The precedence level determines the order in which the operators will be evaluated. Operators at low precedence levels are evaluated before operators at higher levels. Operators in the same precedence level are evaluated according to the association, either right to left (R to L) or left to right (L to R). Operators in the same precedence level are not listed in any particular order.
