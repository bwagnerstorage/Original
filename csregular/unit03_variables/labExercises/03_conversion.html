Conversion

Directions

Write a program that declares seven variables of type double named inch, yard, mile, centimeter, meter, kilometer, and lightYear respectively.

Assign each variable with the following values:

Variable Value
inch
2.54
yard
0.9144
mile
1.609
centimeter
0.3937
meter
1.094
kilometer
0.6214
lightYear
9.464E12

Write the contents of each variable to the console window using the format in the Sample Run below. Make sure you include the label at the end of each line.

Source File

Conversion.java

Sample Run

       Units and Conversion Factors
----------------------------------------------
1 inch = 2.54 centimeters
1 yard = 0.9144 meters
1 mile = 1.609 kilometers
1 centimeter = 0.3937 inches
1 meter = 1.094 yards
1 kilometer = 0.6214 miles
1 light year = 9.464E12 kilometers
