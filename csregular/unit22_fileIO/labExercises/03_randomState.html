RandomStates

Directions

Write a Java program that reads the names of the fifty states from a data file, stores them in an array, and displays them on the screen. Then have the program display a random state on the screen every time the <Enter> key is pressd.

Programing Notes

Use the following code when displaying the random state names:

Scanner keyboard = new Scanner(System.in);
while(true)
{
   // your code goes here


   keyboard.nextLine();   // user must press <Enter> key to continue
}

Placing true in the condition section of the while creates an infinite loop.
