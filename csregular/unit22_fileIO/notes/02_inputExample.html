File Input Examples
What You Will Learn
  • how to use Scanner class to read data from a file

Sample 1 - Read Integers

import java.util.*;  // needed for Scanner
import java.io.*;    // needed for File

public class Sample1
{
    public static void main(String[] args) throws IOException
    {
        int num = 0;
        Scanner scan = new Scanner(new File("Sample1.dat"));
        
        while(scan.hasNextInt())
        {
            num = scan.nextInt();
            System.out.print(num + " ");
        }
        System.out.println();
        
    }
}

Data File - Sample1.dat

23 83 12 98 77 64 53 87 93 39 90 69 32 31 87 3 24 8 21 19 72 36 23 11 9 72 80 26 50 5 13 88

Sample Output

23 83 12 98 77 64 53 87 93 39 90 69 32 31 87 3 24 8 21 19 72 36 23 11 9 72 80 26 50 5 13 88

 

Sample 2 - Read Strings

import java.util.*;  // needed for Scanner
import java.io.*;    // needed for File

public class Sample2
{
    public static void main(String[] args) throws IOException
    {
        String str = "";
        Scanner scan = new Scanner(new File("Sample2.dat"));
        
        while(scan.hasNext())
        {
            str = scan.next();
            System.out.println(str);
        }
        System.out.println();
        
    }
}
