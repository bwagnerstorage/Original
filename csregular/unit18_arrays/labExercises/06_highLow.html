HighLow

Directions

Write a program that generates 10 random real numbers in the range of 0-99, calculated to two decimal places, and stores them in an array. The program should then calculate the average of the 10 numbers. All of the numbers in the array that are less the average should be stored in a second array and all of numbers that are greater than the average should be stored in a third array. The program should finish by printing a summary report which includes the original list of numbers, the average, the list of numbers less than the average, and the list of numbers greater than the average.
