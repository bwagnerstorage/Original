Assignment

In this assignment you are going to implement two classes: an Assignment class and an AssignmentManager class. The AssignmentManager class will be a client of the Assignment class. It will also contain the main method for the program.

Replit Users: If you are using Replit you will need to rename the AssignmentManager class to Main. Replit requires that your program have a Main class.

Directions

Assignment class

The Assignment class is going to store information about a school assignment. This information includes the assignment name, assignment type (daily work, quiz, test), and the assignment score. The class will also have the ability to convert a numerical score into a letter grade(A,B,C,D,F).

Instance Variables

  1. Define three instance variables named name, type, and score. Name and type are of type String and score is of type int.

Constructors

  1. Implement two constructors. A default constructor and second constructor for assigning initial values to the instance variables specified by the client.

Accessor Methods

  1. Implement an accessor method for each instance variable.

Mutatator Methods

  1. Implement a mutator method for each instance variable.

Other Methods

  1. Implement a method named getLetterGrade that returns a string representation of a numerical score. The method should be able to determine a letter grade for an assignment according to the following grading scale:
