Programming Notes

  • There should be 10 x 10 squares.
  • Each square should have the dimensions 40 x 40 pixels.
  • Use nested for loops.
  • Graphic's class setColor method
    page.setColor(color);
  • Graphic's class fillRect method
    page.fillRect(x, y, 40, 40);
  • You can use the following code to alternate between the colors black and white.
    Assume a Color variable named color has been declared.
    if(color == Color.white)  
   color = Color.black;
else
   color = Color.white;
