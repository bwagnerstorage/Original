Create a Jeroo Assignment

Directions

This is a group activity for groups of sizes 1 to 3 people. Your task is to create a Jeroo assignment that allows other students to practice a concept that you have learned while using Jeroo. Your assignment could cover somethinng as simple as the basic Jeroo action methods (hop, pick, toss, etc.) or more advanced topics like user-defined methods (turnAround, disableNet, etc.), control-structures(while, if, if/else) and sensor methods(isWater, isNet, isJeroo, etc.), logical operators (! - not, && - and, || - or), or recursion.

Project Requirements

Your project should include the following:

  • A working model of the assignment with its code and island file. (Your assignment is not required to have an island.
  • An assignment description that resembles the format I used in creating the assignments on my website. This can be written using word, google docs, or html.
  • The assignment description should also include a listing of the programmming concepts your assignment is targeting.
  • Screenshots of the island that provide a visual description of what the island should look like at different stages of the program.
  • At the top of the assignment page include a section that lists all of the students in your group.

Submitting Assignment

You will turn in your assignment by creating a folder on one of your group members' google drive and then share that folder with me at bwagner@g.dentonisd.org. The folder name should use the following format: Class Block_Student1 - First Initial LastInitial_Student2 - First Initial Last Initial_Student3 - First Initial Last Initial. Example folder names: 1_AM_CR_KB, B3_TN_DW
