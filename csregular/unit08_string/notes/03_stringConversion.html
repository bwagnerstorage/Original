String Conversion
What You Will Learn
  • Use Integer class's parseInt method to convert a string to an int
  • Use Double class's parseDouble method to convert a string to a double

Integer Class

It is sometimes necessary in your programs to convert a string to another type such as int or double. Java provides a class named Integer that has methods for this purpose. As an example, suppose you want to convert the string "35" to an integer. This can be done with the following code segment:

String age = "35";
int num = Integer.
parseInt
(age);  
// the string stored in age is

                                  
// converted to an integer value

The code uses the Integer class's parseInt method to convert the string "35" to 35. Once converted it is assigned to the integer variable num.

Double Class

The same conversion can be applied to doubles as well using the Double class's parseDouble method. Refer to the following code segment:

String taxrate = "8.75";
double num = Double.
parseDouble
(taxrate);  
// the string stored in taxrate is

                                           
// converted to a double

The string "8.75" is converted to 8.75.

Caution: The parameter of the parseInt or parseDouble methods must be a digit character or your program will terminate with a run-time error. Example:

int num = Integer.parseInt("A");      
// error

This code will cause an error because the string "A" is not a numeric value, therefore the conversion cannot take place.
