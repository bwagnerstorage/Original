Replace

Directions

Write a Java program that allows a user to enter a string from the keyboard that contains the letter "a". Your program should replace the "a" with an asterisk "*". For example, if the input string is "shark" your program should display the string "sh*rk". If an "a" occurs more than once within the input string only the first occurrence of the letter "a" should be replaced with an asterisk "*".

You will need to use the String class's indexOf and substring methods in your implementation.
