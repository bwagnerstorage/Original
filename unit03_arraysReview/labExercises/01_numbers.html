Numbers

Directions

Write a program that creates 20 random numbers in the range of -100 to 100 and stores them in an array. Then copy all of the positive numbers from the array into a second array and all of the negative numbers into third array. Display all three arrays on the screen like the sample output shown below.

Programming Note

Java provides two methods for creating random numbers. The first method uses the Random class and the second method uses the Math class. With the release of Java 5, the random number generator used by the Math class was improved, therefore the AP development committee opted to use it on the AP Exam instead of the Random class. The main benefit of the Math class is that you do not have to create an extra object inorder to create a random number since the random method is declared static.

Math class

Math.random() - the Math class's random method returns a double value greater than or equal to 0.0 and less than 1.0.

Refer to the sample below for a demonstration of how to implement these two methods.

import java.util.Random;   // needed for Random class

public class RandomSamples
{
    public static void main(String[] args)
    {
        
        /**********************************************************/
        /*                      Random Class                      */
        /**********************************************************/
        
        Random rg = new Random( );

        int n1 = rg.nextInt(20);                   // range 0..19
        int n2 = rg.nextInt(10) + 40;              // range 40..50
        int n3 = rg.nextInt(30) - 30;              // range -30..0
        
        System.out.println("Range 0..19  -> " + n1);
        System.out.println("Range 40..50  -> " + n2);
        System.out.println("Range -30..0 -> " + n3);
        System.out.println();
        
        /**********************************************************/
        /*                       Math Class                       */
        /**********************************************************/
        
        int n4 = (int)(Math.random() * 19);        // range 0..19
        int n5 = (int)(Math.random() * 10) + 40;   // range 40..50
        int n6 = (int)(Math.random() * 30) - 30;   // range -30..0

        System.out.println("Range 0..19   -> " + n4);
        System.out.println("Range 40..50  -> " + n5);
        System.out.println("Range -30..0  -> " + n6);       

    }
}

Modifications

Methods

  1. Implement populateNums method. This method populates the nums array with random numbers in the range of -100 to 100.
  2. Implement populatePositives method. This method populates the positives array with all the positive numbers in the nums array. The variable p is updated to reflect the logical size of positives array.
  3. Implement populateNegatives method. This method populates the negatives array with all the negative numbers in the nums array. The variable n is updated to reflect the logical size of negatives array.
  4. Implement print method. This method, given an array and its logical size, prints all the values of the array from index 0 to its logical size.
  5. Implement printArrays method. This method prints all three arrays with appropriate labelling.
