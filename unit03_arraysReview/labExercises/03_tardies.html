Tardies

Directions

A teacher in your school has asked you to write a program to track the number of tardies students in her class have accrued during the semester. The program should begin by allowing the teacher to enter student information from the keyboard. This information should include the student's last name, first name, and number of tardies. The program should allow for a maximum of 50 students, however the teacher should be allowed to enter less than that number. Once the data entry process is complete, the program should print a summary of class tardies. This summary should include the total number of tardies for her class and a list of all students in the database. Refer to the the sample output below.

Class Decomposition

The problem has been decomposed into 2 classes: Student and Tardies. The Student class is responsible for storing the information for a single student. This class has been provided for you with appropriate instance variables, constructors, accessor methods, mutator methods, and a toString method. The Tardies class is a client of the Student class. It is responsible for maintaining a list of students, entering and managing the student information, including tardies, and for displaying the summary report.

Copy the following code into a file named Student.java

public class Student
{
    // Instance Variables
    private String lastName;
    private String firstName;
    private int tardies;
    
    // Default Constructor
    public Student()
    {
        lastName="";
        firstName="";
        tardies=0;
    }
    
    // Second Constructor
    public Student(String l, String f, int t)
    {
        lastName = l;
        firstName = f;
        tardies = t;
    }
    
    // Accessor Methods
    public String getLastName()
    {
        return lastName;
    }
    public String getFirstName()
    {
        return firstName;
    }
    public int getTardies()
    {
        return tardies;
    }
    
    // Mutator Methods
    public void setLastName(String l)
    {
        lastName = l;
    }
    
    public void setFirstName(String f)
    {
        firstName = f;
    }
    
    public void setTardies(int t)
    {
        tardies = t;
    }
    
    // toString Method
    public String toString()
    {
        String str;
        str = "First Name = " + firstName + "\n" +
              "Last Name  = " + lastName + "\n" +
              "Tardies    = " + tardies + "\n";
              
        return str;
    }
}
