Directions

Write the functions described below. All of the functions should be in the file main.py. At the top of each function display a banner that identifies which function is being displayed. For example:

Function 1

Write a function that prompts a user for his/her name and favorite color then displays the information using the following format: [name] 's favorite color is [color].

An apostrophe and a "s" ('s) should be appended to the person's name.

Sample Run 1

***********
 Function 1
***********
Enter your name -->Steve
Enter your favorite color -->purple

Steve's favorite color is purple

Function 2
