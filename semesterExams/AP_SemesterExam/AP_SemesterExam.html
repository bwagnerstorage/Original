Problem 2

Name:Problem2
Topics: Classes, Loops, Arrays
Directions:Your school band is hosting a garage sale on Saturday and they need a program that will help them track the items donated to the garage sale. The program includes threes classes - the Manager class, GarageSale class, and the GarageSaleItem class. You will only be implementing the GarageSale class. You do not need to modify the Manager class or the GarageSaleItem class.

The program allows the user to enter in garage sale items into a database. When an item is sold the user is able to select the item and mark it with its sale price. The program includes a summary report which includes a list of all garage sale items in the database along with their sale price. The report also prints three totals at the bottom of the report - the total number of items donated, total number of items sold, and total amount made from garage sale.

GarageSale class

The GarageSale class maintains an array called itemList for storing and managing the GarageSaleItems.

You need to implement the following methods:

1. Default Constructor - this constructor should instaniate an array and store its reference in the instance variable itemList. It should also set the logical size of the array to 0.

2. The addItem Method - this method, given a GarageSaleItem, should add the item to the itemList and increment the array's logical size.

3. The getItem Method - this method, given an integer index, should return the GarageSaleItem from the itemList that corresponds to the specified index number.

4. The getTotalItemsCollected Method - this method should return the logical size of the array. The logical size is current number of GarageSaleItems in the array.

5. The getTotalItemsSold Method - this method should calculate and return the total number of items sold from the garage sale. Here is its pseudocode:

create count variable and set it to zero

while the number of garageSaleItems has not been reached
     get an Item from the list
     if the PriceSold of the item is not equal to zero
          add one to count
return count

7. The getTotalSales method - this method should calculate and return the total amount of sales from the garage sale. Here is its pseudocode:

create total variable and set it to zero

while the number of garageSaleItems has not been reached
     get an Item from the list 
     if the PriceSold of the item is not equal to zero
          add the PriceSold to the total
return total
