Using Enhanced for Loop with 2D Arrays
What You Will Learn
  • How to use an enhanced for loop to access elements within a 2D Array.

Enhanced for Loop

Recall that in Java a two-dimensional array is actually an array that contains other arrays.

View the following illustration that demonstrates how the enhanced for loop can take advantage of this property.

Example 1

int[][] matrix = {{84, 27, 91},   // 1st array
                  {77, 42, 65},   // 2nd array
                  {38, 73, 15}};  // 3rd array

for(int[] arr : matrix)
{ 
   for(int num : arr)
   {
      System.out.print(num + " ");
   }
   System.out.println();
}

Example 2

String[][] table = {{"red", "green", "blue"},        // 1st array
                    {"orange", "yellow", "black"}};  // 2nd array

for(String[] list : table)
{ 
   for(String str : list)
   {
      System.out.print(str.substring(0, 1) + " ");  // print 1st letter
   }
   System.out.println();
}
