LessThan100

Directions

Write a program that allows the user to input the dimensions of a two-dimensional array, also called a matrix, then creates a matrix of the specified size filled with random numbers in the range of 1 to 200. The program should finish by printing the matrix and all row-column combinations of the cells that contain values less than 100.
