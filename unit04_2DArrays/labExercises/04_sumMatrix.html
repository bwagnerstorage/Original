SumMatrix

Directions

Write a program that prompts the user for the dimensions of a two-dimensional array. It then creates an array with the specified size and fills each cell with random integers in the range of 0 to 9. The program should then print the two-dimensional array and the sum of each row and column as shown in the sample output below.

To do this assignment you will need three sets of nested for loops. One set to create the matrix, another set to sum the rows and print the matrix and the row sums, and a third set to sum the columns and print the column sums below the matrix.
