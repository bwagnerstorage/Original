SeatingChart

Workspace: SeatingChart

Directions

Write a program that allows a teacher to create and store students in a seating chart. A student is represented by the following class:

public class Student
{
    // instance variables
    private String name;
    private String id;

    // default constructor
    public Student()
    {
        name = "vacant";
        id = "";
    }

    // another constructor
    public Student(String n, String i)
    {
        name = n;
        id = i;
    }

    // mutator methods
    public void setName(String n)
    {
        name = n;
    }

    public void setID(String i)
    {
        id = i;
    }

    // accessor methods
    public String getName()
    {
        return name;
    }

    public String getID()
    {
        return id;
    }

    // toString method
    public String toString()
    {
        return String.format("%-10s", name);
    }
}

The seating chart is represented in the class SeatingChart as a 2D array of Student objects:

private Student[][] chart;

You will be writing methods in the SeatingChart class.

Method addStudent

Write method addStudent. Given a Student object and a seat location addStudent adds the Student to the seating chart at a specified location. If the specified location is a valid location the Student is added to the chart and the method returns true; otherwise the Student is not added to the chart and the method returns false.

Method swapSeats

Write method swapSeats. Given two seat locations swapSeats swaps seats of two students in the seating chart. If both seat locations are valid the Students are swapped and the method returns true; otherwise the Students are not swapped and the method returns false.

Method scrambleChart

Write method scrambleChart. scrambleChart creates a new 2D array of Student objects by taking all the students from the seating chart and placing them in a new chart at random locations. Any seats not assigned to a student should be assigned a default Student object which, when displayed, will print the word "vacant". A seat is empty in the new chart if the array cell contains the keyword null.
