Frog

Directions

A Frog behaves like a Bug in every way, except that it is moves or jumps two cells whenever possible and turns like a Bug otherwise. To access a location in the grid two positions away from a frog's current location use the Location class's getAdjacentLocation method twice.

One appropriate to simulate this behavior is to do the following within the move method: 
if frog can jump two spaces then 
   jump
else
   super.move()   // move like a bug

Implement the Frog class then use the FrogRunner class below to test your new class.

Copy the following code into a file named FrogRunner.java.

import info.gridworld.grid.*;
import info.gridworld.actor.*;
import java.awt.Color;

public class FrogRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();

        Frog frog = new Frog();

        world.add(new Location(7,5), frog);
        world.show();
    }
}
