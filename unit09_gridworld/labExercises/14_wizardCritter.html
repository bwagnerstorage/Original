WizardCritter

A Critter is an Actor. A Critter is more advanced than a Bug in that it has the ability to interact with other actors in the grid. This is accomplished by redefining its act method.

Below is the Critter class's implementation of the act method.

public void act()
{
     if (getGrid() == null)
         return;
     ArrayList<Actor> actors = 
getActors()
;
     
processActors(actors)
;
     ArrayList<Location> moveLocs = 
getMoveLocations()
;
     Location loc = 
selectMoveLocation(moveLocs)
;
     
makeMove(loc)
;
}

A Critter acts by calling the following methods

  • getActors - this method gets a list of actors occupying the locations in the 8 compass directions surrounding the Critter.
  • processActors - this method is where the Critter processes (interacts with) the actors returned by getActors. A Critter processes actors be eating any actor that is not a rock or another Critter.
  • getMoveLocations - this method gets a list of unoccupied locations in the 8 compass directions surrounding the Critter.
  • selectMoveLocations - this method selects at random one of the locations returned by getMoveLocations.
  • makeMove - this method moves the Critter to the selected location returned by selectMoveLocation.

Directions

A WizardCritter is a Critter with the power to transform other actors into rocks. A WizardCritter gets actors like a normal Critter. When the WizardCritter encounters another actor within its neighborhood it transforms it into a rock and changes its color to the WizardCritter's color. The WizardCritter has no affect on other WizardCritters.

After transforming the actors around it, a WizardCritter apparates or teleports to a randomly selected empty location somewhere within the grid.

A WizardCritter is instantiated with a color value specified in a constructor's parameter list.


Implement the WizardCritter class then use the WizardCritterRunner class below to test your new class.

Copy the following code into a file named WizardCritterRunner.java.

import info.gridworld.grid.*;
import info.gridworld.actor.*;
import java.awt.Color;

public class WizardCritterRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();
        world.add(new Location(7, 8), new Rock());
        world.add(new Location(3, 3), new Rock());
        world.add(new Location(3,6), new Bug());
        world.add(new Location(6,2), new Bug());
        world.add(new Location(8,1), new Bug());
        world.add(new Location(1,2), new Critter());
        world.add(new Location(4,7), new Critter());
        world.add(new Location(2, 8), new Flower(Color.BLUE));
        world.add(new Location(5, 5), new Flower(Color.PINK));
        world.add(new Location(4, 4), new WizardCritter());
        world.add(new Location(9, 9), new WizardCritter());
        world.show();
    }
}
