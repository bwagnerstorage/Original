RandomBug

Directions

A RandomBug is a Bug. A RandomBug is different from a Bug in three ways.

  1. A RandomBug can move to any location within the grid chosen at random. Inorder to select a random location within the boundaries of the current grid you will need to use the grid methods getNumRows and getNumCols.

    The following code can be used to create a random location within the grid.

    int r = (int)(Math.random() * gr.getNumRows());
int c = (int)(Math.random() * gr.getNumCols());
Location next = new Location(r, c));

  2. A RandomBug does not leave flowers when it moves.

  3. A RandomBug will replace any object, which occupies the randomly selected location.

    Override the canMove method so that always returns true.

Implement the RandomBug class then use the RandomBugRunner class below to test your new class.

import info.gridworld.actor.*;
import info.gridworld.grid.*;
import java.awt.Color;
import java.util.*;

public class RandomBugRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();

        RandomBug randomBug = new RandomBug(Color.YELLOW);
        Bug bug = new Bug();
        world.add(new Location(5, 5), randomBug);
        world.add(new Location(2, 2), bug);
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.add(new Rock());
        world.show();
    }
}
