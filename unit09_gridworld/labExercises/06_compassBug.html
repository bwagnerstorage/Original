CompassBug

Directions

A CompassBug is a special BoxBug. A CompassBug behaves like a BoxBug in every way, except that it changes color with each major compass direction. As the bug moves NORTH it becomes RED, going EAST is becomes GREEN, going SOUTH it becomes BLUE and going WEST it becomes BLACK. The CompassBug accomplishes this by overriding the act method.

Use the Location class's directional constants in your implementation.
Example: Location.NORTH

Implement the CompassBug class then use the CompassBugRunner class below to test your new class.

import info.gridworld.actor.*;
import info.gridworld.grid.*;
import java.awt.Color;
import java.util.*;

public class CompassBugRunner
{
    public static void main(String[] args)
    {
        ActorWorld world = new ActorWorld();
		
        CompassBug bug1 = new CompassBug(6);		
        CompassBug bug2 = new CompassBug(3);
		
        world.add(new Location(8, 2), bug1);
        world.add(new Location(6, 5), bug2);
        world.show();
    }
}
