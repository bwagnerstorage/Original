JavaFX Lab Exercise 4.2

BorderPane

The BorderPane layout pane provides five regions in which to place nodes: top, bottom, left, right, and center. The regions can be any size. If your application does not need one of the regions, you do not need to define it and no space is allocated for it.

A border pane is useful for the classic look of a tool bar at the top, a status bar at the bottom, a navigation panel on the left, additional information on the right, and a working area in the center. If the window is larger than the space needed for the contents of each region, the extra space is given to the center region by default.



Lab Exercise

In this exercise, we will use a BorderPane as our base layout, then we will add other panes to the different regions of the BorderPane to better refine the placement of the UI controls. When complete your GUI window should look like the following:





  1. In the project JavaFxExercise, double-click on the file FXML_BorderPane.fxml to open it in Scene Builder.

  2. Notice that the scene graph hierarchy already contains an BorderPane at its root level.

  3. From the Controls panel, drag-and-drop a MenuBar control onto the Top region of the BorderPane. By default, the menubar comes with three menus included.

  4. From the Preview menu select Show Preview in Window. Resize the window. Notice that the menubar expands horizontally to fill the extra space.

  5. From the Containers panel, drag-and-drop a ScrollPane (empty) onto the Left Region of the BorderPane. A ScrollPane provides both horizontal and vertical scroll bars for data that cannot be viewed within the current window size.

  6. In the Layout panel for the ScrollPane, make the following changes:

    • Set Min Height value to 80.

    • Set Pref Width value to 150.

    • Set Pref Height value to 245.

  7. From the Containers panel, drag-and-drop an Accordion pane on top of the ScrollPane.

  8. From the Preview menu select Show Preview in Window. Click on one of accordion toggles to expand it. Resize the window vertically until a vertical scroll bar appears.

  9. From the Containers panel, drag-and-drop a TabsPane onto the Center Region of the BorderPane.

  10. From the Containers panel, drag-and-drop a BorderPane onto the Bottom Region of the main BorderPane.

  11. From the Controls panel, drag-and-drop a Button control onto the Left region of this second BorderPane.

  12. From the Controls panel, drag-and-drop a Button control onto the Right region of this second BorderPane.

  13. From the Controls panel, drag-and-drop a Label control onto the Left region of this second BorderPane. Change the text of this label to "Status Bar" and change its font size to System 18px.

  14. From the Preview menu select Show Preview in Window. Resize the window and notice how the different layout panes respond.
