Bubble Sort

The bubble sort is the oldest and simplest sort in use. Unfortunately, it's also the slowest. The bubble sort works by comparing the first two elements in the list. If the second is larger than the first, the two are swapped. It then compares the second and third item in the list, and these too are swapped if necessary. When you are done with one pass, you have bubbled the largest item to the end of the list. The next time through the list, you compare item 0 through n - 1. That is, you start over, bubbling up the next-largest item in the list, comparing it in turn with every item except the last item. When you are done with the second pass, the largest and second-largest items are in place.

Bubble Sort Efficiency

The bubble sort is generally considered to be the most inefficient sorting algorithm in common usage. Under best case conditions (the list is already sorted), the bubble sort can approach a constant O(n) level of complexity. Average case is O(n2).

While the insertion and selection sorts also have O(n2) complexities, they are significantly more efficient than the bubble sort.

Pros: Simplicity and ease of implementation.
Cons: Horribly inefficient.

Bubble Sort algorithm (Version 1)

void bubbleSort(int numbers[])
{
  //  Sorts list in ascending order
  int i, j, temp;

  for (i = numbers.length - 1; i >= 1; i--)
  {
     for (j = 1; j <= i; j++)
     {
        if (numbers[j-1] > numbers[j])
        {
	        // swap
            temp = numbers[j-1];
            numbers[j-1] = numbers[j];
            numbers[j] = temp;
        }
     }
  }
}

Bubble Sort algorithm (Version 2) - more efficient version

public void bubbleSort(int[] numbers)
    {

        //  Sorts list in ascending order, this version is 
        //  more efficient than version 1 because if will stop
        //  as soon as it makes a pass through the list and does not
        //  perform a swap. (i.e. a list that is already sorted)
        
        int i, temp;
        boolean done;
        do
        {
           done = true;
           for(i = 0; i < numbers.length-1; i++)
           {
               if (numbers[i] > numbers[i + 1])
               {            
                   // swap
                   temp = numbers[i];
                   numbers[i] = numbers[i+1];
                   numbers[i+1] = temp;
                   done = false;             // made a swap
               }            
           }        
        } 
        while (!done);
    }
