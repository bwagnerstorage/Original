Lab Exercise

  1. Open Notepad and create a new document named 11_1_links1.html and save it to your HTML folder.

  2. Give the document the title: 11_1_links1.

  3. Create a second document named 11_1_links2.html and save it to your HTML folder.

  4. Give the document the title: 11_1_links2

  5. Create a heading on the first page labelled First Page.

  6. Create a heading on the second page labelled Second Page.

  7. Create a link on the first page labelled Second Page that when clicked will open the second page.

  8. Right-click on the smiley face image and save it to your HTML folder with the name smileyFace.png.

  9. Create an image link on the second page using the smiley face image that when clicked will take you back to the first page.
