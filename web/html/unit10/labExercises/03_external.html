10-3 HTML External CSS

Chapters

  • HTML CSS

Lab Exercise

  1. Open Notepad and create a new document named 10_3_external.html and save it to your HTML folder.

  2. Give the document the title: 10_3_external.

  3. Add the following html to your document.


    <!DOCTYPE html>
    <html>
    <head>
    </head>
    <body>
    <h1>Pool Rules</h1>
    <p>Lifeguard not on duty. Swim at your own risk.</p>
    <p>Children under the age of 16 must be accompanied by an adult.</p>
    <p>No Diving.</p>
    </body>
    </html>

  4. In Notepad create a new document by selecting File -> New.

  5. Save the document in your HTML folder with the name 10_3_external.css.

  6. Add the following line of code to your html document between the header tags.

    <link rel="stylesheet" href="10_3_external.css">

  7. Make the following modifications using external styling:

    • Change the body element's background color to seashell.

    • Change the heading element's text color to purple.

    • Change the heading element's font to veranda.

    • Increase the heading element's font size to 400%.

    • Change each of the three paragraph element's text color to saddlebrown.

    • Increase the paragraph element's font size to 200%.
