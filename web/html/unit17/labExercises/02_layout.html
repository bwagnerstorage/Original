Lab Exercise

  1. Open Notepad and create a new document named 17_2_layout.html and save it to your HTML folder.

  2. Give the document the title: 17_2_layout.

  3. Use the HTML 5 layout elements header, footer, nav, section, and aside to create the layout shown below.

  4. Use internal styling to format the elements.

  5. Use any color you like for the different elements.

  6. Use the h3 tag to label each section.

  7. The header and footer elements should have a height of 100px.

  8. The nav and aside elements should have a width of 150px. The nav element should float left and the aside should float right.

  9. The section element should include the following attribute: overflow:hidden.

  10. Important: the section element needs to be declared after the aside element within the body element.
