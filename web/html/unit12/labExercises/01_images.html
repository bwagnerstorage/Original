12-1 HTML Images

Chapters

  • HTML Images

Lab Exercise

  1. Open your HTML folder. Create a new folder named images. From now on any images you display on your webpages should be placed inside this folder.

  2. Open Notepad and create a new document named 12_1_images.html and save it to your HTML folder.

  3. Give the document the title: 12_1_images.

  4. Create a heading labelled The ENIAC.

  5. Add the following information to the web page in a paragraph tag:

    The ENIAC built during World War II was the first programmable general-purpose electronic computer. It was 100 feet long and used more than 17,000 vacuum tubes. It was programmed by placing patch wires into plugboards. Click on the following link to learn more about the ENIAC.

  6. Make the word ENIAC in the last sentence a link to the following website: http://the-eniac.com/. Use the target attribute so the page will open in a new browser tab.

  7. Right-click on the image below and save it to your images folder with the name ENIAC.jpg.

  8. Display the image on the page so that the image appears to the right of the text. To do this you will need to use inline styling of the img tag and the float attribute with a value of right. Make sure you place the img tag inside the paragraph tag before the text.

  9. Use inline styling to specify the width and height of the image. The width is 702px and the height is 478px.
