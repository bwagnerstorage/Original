    Java While Loop

    Java while loops can be classified into two categories: counter-controlled loops and sentinel-controlled loops.

    Counter-Controlled Loop

    A counter-controlled loop is a loop that repeats a predetermined number of times. The condition of this type of loop is controlled by a counter variable. A counter is a variable that keeps count of the number of times a loop is executed. Look at the following example of a counter-controlled loop.

    int counter = 1;

    while(counter <= 10)
    {
    System.out.println(counter);
    counter++;
    }

    Sentinel-Controlled Loop

    A sentinel-controlled loop is a loop that executes an undetermined number of times. To stop the loop, a user enters a value, called a sentinel, which causes the loop's condition to become false. Look at the following example of a sentinel-controlled loop.

    Scanner sc = new Scanner(System.in);
    String ans = "";

    while(!ans.equals("stop"))
    {
    ans = sc.nextLine();
    }
