3-1 HTML Attributes

Chapters

  • HTML Attributes

Lab Exercise

  1. Open Notepad and create a new document named 3_1_attributes.html and save it to your HTML folder.

  2. Add the following code right above the body element: 

    <head>
   <title>3_1_attributes</title>
</head>

  3. Create a html document that includes the following:

    • The heading My Favorite Websites.

    • Use the anchor tag <a> to create links to three of your favorite websites. Use the href attribute so that when a link is clicked it will open the designated website.

    • Include the paragraph tag <p> around each anchor tag so that each link will appear on a separate line.

  4. Right-click on the picture of the puppy and save the image to your HTML folder with the name puppy1.jpg.

  5. Determine the size of the image by following these steps:

    • Open your HTML folder.

    • Right-click on the puppy image icon and select Properties.

    • Select the Details tab.

    • Under the Image section you will find the dimensions of the image.

  6. Use the image tag <img> so that the puppy image is displayed on this page.

  7. Use the src attribute to specify the name and location of image file.

  8. Use the width and height attributes to specify the size of the image.
