Area

Directions

A set of classes is used to calculate the area of various geometric shapes. The Rectangle class calculates the area of a rectangle and the Circle class calculates the area of a circle. To provide a standard functionality between the two classes, both classes implement the Shape interface.

Copy the following code into a file named Shape.java

public interface Shape
{
    double getArea();
}

  1. Write the complete class declaration for the class Rectangle.

    Copy the following code into a file named Rectangle.java.

    public class Rectangle implements Shape
{

}

    - Create an instance variable of type double named length.

    - Create an instance variable of type double named width.

    - Implement the method getArea using the following mathematical formula: area = length x width

    - Implement a constructor that takes two parameters that indicate the length and width of the rectangle.

  2. Write the complete class declaration for the class Circle.

    Copy the following code into a file named Circle.java.

    public class Circle implements Shape
{

}

    - Create an instance variable of type double named radius.

    - Implement the method getArea using the following mathematical formula: area = pi x radius2

    - Implement a constructor that takes a parameter that indicates the radius of the circle.

  3. Implement the method printArea in the CalculateArea class.

    Copy the following code into a file named CalculateArea.java.

    public class CalculateArea
{

    {
        Shape rectangle = new Rectangle(10, 20);
        Shape circle = new Circle(5);

        System.out.println("Circle");
        System.out.println("------");
        printArea(circle);    
    }
    
    // prints the area of the given shape
    public void 
    printArea

    This method should print the area of the parameter shape in the following format

    Area = 57.5

    Optional: use printf to round area to one decimal place.
