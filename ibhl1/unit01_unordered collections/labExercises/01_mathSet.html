MathSet

The union of two sets A and B is a set that contains all of the items in A and all of the items in B.
The intersection of two sets A and B is a set of items in A that are also items in B.
The difference of two sets is the set of items in A that are not also items in B.
The set B is a subset of set A if and only if B is an empty set or all of the items in B are also in A.
