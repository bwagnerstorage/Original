Problem 5

Write a program that can display a date in three different formats: Slash Format, Dash Format, and Word Format. Each format should be implemented in their corresponding method.

Slash Format

Implement the SlashFormat method. This method should print a Date object on the screen in the following format:

month/day/year (Ex. 3/12/1964)

The method should use the Date object, date, that is passed to the method through the parameter list.

 

Dash Format

Implement the DashFormat method. This method should print a Date object on the screen in the following format:

month-day-year (Ex. 3-12-1964)

The method should use the Date object, date, that is passed to the method through the parameter list.

 

Word Format

Implement the WordFormat method. This method should print a Date object on the screen in the following format:

month day, year (Ex. March 12, 1964)

The method needs to convert the month from a number format into word format (Examples: 1 = January, 2 = February, 3 = March, etc.)

The method should use the Date object, date, that is passed to the method through the parameter list.

public class Date
{
   // Instance Variables
   private int Month;
   private int Day;
   private int Year;
   
   // Default Constructor
   public Date()
   {
      Day = 0;
      Month = 0;
      Year = 0;
   }   
   // Second Constructor
   public Date(int m, int d, int y)
   {
      Day = d;
      Month = m;
      Year = y;
   }
   
   // Accessor Methods
   public int getDay()
   {
      return Day;
   }
   
   public int getMonth()
   {
      return Month;
   }
   
   public int getYear()
   {
      return Year;
   }
   
   // Mutator Methods
   public void setDay(int d)
   {
      Day = d;
   }
   
   public void setMonth(int m)
   {
      Month = m;
   }
   
   public void setYear(int y)
   {
      Year = y;
   }
   
   // toString Method
   public String toString()
   {
      String str;
      str = Month + "/" + Day + "/" + Year;
      return str;
   }
}
